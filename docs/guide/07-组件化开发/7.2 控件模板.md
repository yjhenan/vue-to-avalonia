## 7.2 控件模板

### 7.2.1 ControlTemplate vs Vue slot

**Vue的slot：**

```vue
<!-- Card.vue -->
<template>
  <div class="card">
    <div class="card-header">
      <slot name="header">
        <!-- 默认内容 -->
        <h3>标题</h3>
      </slot>
    </div>

    <div class="card-body">
      <slot>
        <!-- 默认内容插槽 -->
        内容
      </slot>
    </div>

    <div class="card-footer">
      <slot name="footer">
        <button>确定</button>
      </slot>
    </div>
  </div>
</template>
```

**使用：**

```vue
<Card>
  <template #header>
    <h2>自定义标题</h2>
  </template>

  <p>这是卡片内容</p>

  <template #footer>
    <button @click="handleSave">保存</button>
    <button @click="handleCancel">取消</button>
  </template>
</Card>
```

**Avalonia的ControlTemplate：**

```xml
<!-- Templates/Card.xaml -->
<Styles xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">

  <!-- 定义Card的ControlTemplate -->
  <Style Selector="local|Card">
    <Setter Property="Template">
      <ControlTemplate>
        <Border Background="{TemplateBinding Background}"
                BorderBrush="{TemplateBinding BorderBrush}"
                BorderThickness="{TemplateBinding BorderThickness}"
                CornerRadius="8"
                Padding="15">

          <Grid RowDefinitions="Auto,*,Auto">
            <!-- Header（类似name="header"的slot） -->
            <ContentPresenter Grid.Row="0"
                              Content="{TemplateBinding Header}"
                              Margin="0,0,0,10"/>

            <!-- Body（类似默认slot） -->
            <ContentPresenter Grid.Row="1"
                              Content="{TemplateBinding Content}"/>

            <!-- Footer（类似name="footer"的slot） -->
            <ContentPresenter Grid.Row="2"
                              Content="{TemplateBinding Footer}"
                              Margin="0,10,0,0"/>
          </Grid>

        </Border>
      </ControlTemplate>
    </Setter>
  </Style>

</Styles>
```

**Card控件定义：**

```csharp
// Controls/Card.cs
using Avalonia;
using Avalonia.Controls;

namespace MyApp.Controls
{
    public class Card : TemplatedControl
    {
        // Header属性
        public static readonly StyledProperty<object> HeaderProperty =
            AvaloniaProperty.Register<Card, object>(nameof(Header));

        // Content属性（默认内容）
        public static readonly StyledProperty<object> ContentProperty =
            AvaloniaProperty.Register<Card, object>(nameof(Content));

        // Footer属性
        public static readonly StyledProperty<object> FooterProperty =
            AvaloniaProperty.Register<Card, object>(nameof(Footer));

        public object Header
        {
            get => GetValue(HeaderProperty);
            set => SetValue(HeaderProperty, value);
        }

        public object Content
        {
            get => GetValue(ContentProperty);
            set => SetValue(ContentProperty, value);
        }

        public object Footer
        {
            get => GetValue(FooterProperty);
            set => SetValue(FooterProperty, value);
        }
    }
}
```

**使用Card：**

```xml
<local:Card Margin="20">
  <!-- Header -->
  <local:Card.Header>
    <TextBlock Text="自定义标题"
               FontSize="18"
               FontWeight="Bold"/>
  </local:Card.Header>

  <!-- Content（默认内容） -->
  <local:Card.Content>
    <TextBlock Text="这是卡片内容"/>
  </local:Card.Content>

  <!-- Footer -->
  <local:Card.Footer>
    <StackPanel Orientation="Horizontal" Spacing="10">
      <Button Content="保存" Command="{Binding SaveCommand}"/>
      <Button Content="取消" Command="{Binding CancelCommand}"/>
    </StackPanel>
  </local:Card.Footer>
</local:Card>
```

### 7.2.2 DataTemplate vs Vue render props

**Vue的render props：**

```vue
<!-- ListView.vue -->
<template>
  <div class="list">
    <div v-for="item in items" :key="item.id">
      <slot name="item" :item="item" :index="index">
        <!-- 默认渲染 -->
        <p>{{ item.name }}</p>
      </slot>
    </div>
  </div>
</template>
```

**使用：**

```vue
<ListView :items="users">
  <template #item="{ item, index }">
    <div class="user-item">
      <img :src="item.avatar" />
      <span>{{ index }}. {{ item.name }}</span>
    </div>
  </template>
</ListView>
```

**Avalonia的DataTemplate：**

```xml
<Window xmlns="https://github.com/avaloniaui">
  <Window.Resources>
    <!-- 默认DataTemplate -->
    <DataTemplate x:Key="DefaultItemTemplate">
      <TextBlock Text="{Binding Name}"/>
    </DataTemplate>

    <!-- 自定义DataTemplate（类似作用域插槽） -->
    <DataTemplate x:Key="UserItemTemplate">
      <StackPanel Orientation="Horizontal" Spacing="10">
        <Image Width="40" Height="40"
               Source="{Binding Avatar}"
               CornerRadius="20"/>
        <TextBlock VerticalAlignment="Center">
          <Run Text="{Binding Index}"/>.
          <Run Text="{Binding Name}"/>
        </TextBlock>
      </StackPanel>
    </DataTemplate>
  </Window.Resources>

  <!-- 使用DataTemplate -->
  <ItemsControl ItemsSource="{Binding Users}"
                ItemTemplate="{StaticResource UserItemTemplate}"/>

</Window>
```

**或者内联定义：**

```xml
<ItemsControl ItemsSource="{Binding Users}">
  <ItemsControl.ItemTemplate>
    <DataTemplate>
      <StackPanel Orientation="Horizontal" Spacing="10">
        <Image Width="40" Height="40"
               Source="{Binding Avatar}"
               CornerRadius="20"/>
        <TextBlock Text="{Binding Name}"
                   VerticalAlignment="Center"/>
      </StackPanel>
    </DataTemplate>
  </ItemsControl.ItemTemplate>
</ItemsControl>
```
