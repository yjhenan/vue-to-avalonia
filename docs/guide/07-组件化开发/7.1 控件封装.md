# 第七章：组件化开发——从Vue组件到UserControl

组件化是现代UI开发的核心思想。在Vue中，你习惯了使用`.vue`单文件组件来封装可复用的UI单元。在Avalonia中，我们使用`UserControl`来实现相同的目的。本章将帮助你掌握Avalonia的组件化开发。

## 7.1 控件封装

### 7.1.1 Vue组件 vs Avalonia UserControl

**Vue单文件组件结构：**

```vue
<!-- UserCard.vue -->
<template>
  <div class="user-card">
    <img :src="user.avatar" :alt="user.name" />
    <div class="user-info">
      <h3>{{ user.name }}</h3>
      <p>{{ user.email }}</p>
      <span class="status" :class="{ active: user.isActive }">
        {{ user.isActive ? '在线' : '离线' }}
      </span>
    </div>
    <button @click="$emit('edit', user)">编辑</button>
  </div>
</template>

<script setup>
import { computed } from 'vue'

const props = defineProps({
  user: {
    type: Object,
    required: true
  }
})

const emit = defineEmits(['edit'])

// 组件内部逻辑
const statusText = computed(() =>
  props.user.isActive ? '在线' : '离线'
)
</script>
```

### 7.1.2 组件通信（Props/Emits vs 依赖属性/路由事件）

**Vue的组件通信：**

```vue
<!-- 父组件 -->
<template>
  <ChildComponent
    :title="parentTitle"
    :items="items"
    @update="handleUpdate"
    @delete="handleDelete"
  />
</template>

<!-- 子组件 -->
<script setup>
const props = defineProps({
  title: String,
  items: Array
})

const emit = defineEmits(['update', 'delete'])

function onClick() {
  emit('update', newValue)
}
</script>
```

**Avalonia的组件通信（依赖属性）：**

```csharp
// Controls/ItemListView.axaml.cs
using Avalonia;
using Avalonia.Controls;
using Avalonia.Interactivity;
using System.Collections.ObjectModel;
using System.Windows.Input;

namespace MyApp.Controls
{
    public partial class ItemListView : UserControl
    {
        // 1. 定义依赖属性（类似props）
        public static readonly StyledProperty<string> TitleProperty =
            AvaloniaProperty.Register<ItemListView, string>(nameof(Title), "默认标题");

        public static readonly StyledProperty<ObservableCollection<Item>> ItemsProperty =
            AvaloniaProperty.Register<ItemListView, ObservableCollection<Item>>(nameof(Items));

        // 2. 定义路由事件（类似emits）
        public static readonly RoutedEvent<RoutedEventArgs> UpdateEvent =
            RoutedEvent.Register<ItemListView, RoutedEventArgs>(
                nameof(Update),
                RoutingStrategies.Bubble);

        public static readonly RoutedEvent<RoutedEventArgs> DeleteEvent =
            RoutedEvent.Register<ItemListView, RoutedEventArgs>(
                nameof(Delete),
                RoutingStrategies.Bubble);

        // 3. CLR属性包装器
        public string Title
        {
            get => GetValue(TitleProperty);
            set => SetValue(TitleProperty, value);
        }

        public ObservableCollection<Item> Items
        {
            get => GetValue(ItemsProperty);
            set => SetValue(ItemsProperty, value);
        }

        // 4. 事件访问器
        public event EventHandler<RoutedEventArgs> Update
        {
            add => AddHandler(UpdateEvent, value);
            remove => RemoveHandler(UpdateEvent, value);
        }

        public event EventHandler<RoutedEventArgs> Delete
        {
            add => AddHandler(DeleteEvent, value);
            remove => RemoveHandler(DeleteEvent, value);
        }

        public ItemListView()
        {
            InitializeComponent();
        }

        // 5. 触发事件（类似emit）
        private void OnUpdateButtonClicked(object? sender, RoutedEventArgs e)
        {
            var eventArgs = new RoutedEventArgs(UpdateEvent);
            RaiseEvent(eventArgs);  // 类似emit('update', data)
        }

        private void OnDeleteButtonClicked(object? sender, RoutedEventArgs e)
        {
            var eventArgs = new RoutedEventArgs(DeleteEvent);
            RaiseEvent(eventArgs);
        }
    }
}
```

### 7.1.3 完整示例：可复用的评分组件

**Vue组件实现：**

```vue
<!-- Rating.vue -->
<template>
  <div class="rating">
    <span
      v-for="star in 5"
      :key="star"
      class="star"
      :class="{ filled: star <= modelValue }"
      @click="updateRating(star)"
    >
      ★
    </span>
    <span class="rating-text">{{ modelValue }}/5</span>
  </div>
</template>

<script setup>
const props = defineProps({
  modelValue: {
    type: Number,
    default: 0,
    validator: (value) => value >= 0 && value <= 5
  },
  readonly: {
    type: Boolean,
    default: false
  }
})

const emit = defineEmits(['update:modelValue'])

function updateRating(star) {
  if (!props.readonly) {
    emit('update:modelValue', star)
  }
}
</script>

<style scoped>
.rating {
  display: flex;
  align-items: center;
  gap: 5px;
}

.star {
  font-size: 24px;
  color: #ddd;
  cursor: pointer;
  transition: color 0.2s;
}

.star.filled {
  color: #ffc107;
}

.star:hover:not(.readonly) {
  color: #ffb300;
}

.rating-text {
  margin-left: 10px;
  color: #666;
}
</style>
```

**RatingControl.axaml.cs：**

```csharp
// Controls/RatingControl.axaml.cs
using Avalonia;
using Avalonia.Controls;
using Avalonia.Input;
using System;

namespace MyApp.Controls
{
    public partial class RatingControl : UserControl
    {
        // ===== 依赖属性 =====

        // Rating属性（类似v-model）
        public static readonly StyledProperty<int> RatingProperty =
            AvaloniaProperty.Register<RatingControl, int>(
                nameof(Rating),
                defaultValue: 0,
                coerce: CoerceRating);

        // IsReadonly属性
        public static readonly StyledProperty<bool> IsReadonlyProperty =
            AvaloniaProperty.Register<RatingControl, bool>(
                nameof(IsReadonly),
                defaultValue: false);

        // RatingChanged事件（类似@update:modelValue）
        public static readonly RoutedEvent<RatingChangedEventArgs> RatingChangedEvent =
            RoutedEvent.Register<RatingControl, RatingChangedEventArgs>(
                nameof(RatingChanged),
                RoutingStrategies.Bubble);

        // ===== CLR属性 =====

        public int Rating
        {
            get => GetValue(RatingProperty);
            set => SetValue(RatingProperty, value);
        }

        public bool IsReadonly
        {
            get => GetValue(IsReadonlyProperty);
            set => SetValue(IsReadonlyProperty, value);
        }

        // ===== 事件 =====

        public event EventHandler<RatingChangedEventArgs> RatingChanged
        {
            add => AddHandler(RatingChangedEvent, value);
            remove => RemoveHandler(RatingChangedEvent, value);
        }

        // ===== 计算属性（用于绑定） =====

        public bool Star1Filled => Rating >= 1;
        public bool Star2Filled => Rating >= 2;
        public bool Star3Filled => Rating >= 3;
        public bool Star4Filled => Rating >= 4;
        public bool Star5Filled => Rating >= 5;
        public string RatingText => $"{Rating}/5";

        // ===== 构造函数 =====

        public RatingControl()
        {
            InitializeComponent();
            DataContext = this;  // 简化绑定

            // 监听Rating变化，更新星星状态
            this.GetObservable(RatingProperty).Subscribe(_ =>
            {
                RaisePropertyChanged(nameof(Star1Filled));
                RaisePropertyChanged(nameof(Star2Filled));
                RaisePropertyChanged(nameof(Star3Filled));
                RaisePropertyChanged(nameof(Star4Filled));
                RaisePropertyChanged(nameof(Star5Filled));
                RaisePropertyChanged(nameof(RatingText));
                // 注意：AvaloniaObject 的 RaisePropertyChanged，非 ReactiveUI
            });
        }

        // ===== 验证逻辑 =====

        private static int CoerceRating(AvaloniaObject obj, int value)
        {
            // 验证范围 0-5
            return Math.Clamp(value, 0, 5);
        }

        // ===== 点击事件 =====

        private void OnStar1Click(object? sender, PointerPressedEventArgs e)
        {
            SetRating(1);
        }

        private void OnStar2Click(object? sender, PointerPressedEventArgs e)
        {
            SetRating(2);
        }

        private void OnStar3Click(object? sender, PointerPressedEventArgs e)
        {
            SetRating(3);
        }

        private void OnStar4Click(object? sender, PointerPressedEventArgs e)
        {
            SetRating(4);
        }

        private void OnStar5Click(object? sender, PointerPressedEventArgs e)
        {
            SetRating(5);
        }

        private void SetRating(int rating)
        {
            if (!IsReadonly)
            {
                Rating = rating;

                // 触发事件
                var args = new RatingChangedEventArgs(RatingChangedEvent, rating);
                RaiseEvent(args);
            }
        }
    }

    // 自定义事件参数
    public class RatingChangedEventArgs : RoutedEventArgs
    {
        public int NewRating { get; }

        public RatingChangedEventArgs(RoutedEvent routedEvent, int newRating)
            : base(routedEvent)
        {
            NewRating = newRating;
        }
    }
}
```

## 7.2 小结

本节学习了Avalonia的UserControl组件化开发：

✅ **UserControl基础**：创建和使用自定义控件
✅ **组件通信**：Props → 依赖属性，Emits → 路由事件
✅ **完整示例**：实现评分组件，包含样式、交互、验证

**Vue组件 vs Avalonia UserControl对照：**

| Vue组件 | Avalonia UserControl | 说明 |
|---------|---------------------|------|
| `.vue`文件 | `.axaml` + `.axaml.cs` | 组件结构 |
| `template` | XAML | UI模板 |
| `script` | ViewModel + Code-behind | 逻辑代码 |
| `style` | Styles | 样式定义 |
| `props` | StyledProperty | 输入属性 |
| `emits` | RoutedEvent | 输出事件 |
| `v-model` | TwoWay绑定 | 双向绑定 |
| `<slot>` | ContentPresenter | 内容分发 |
| 组件注册 | xmlns引用 | 使用组件 |
