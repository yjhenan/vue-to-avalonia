## 3.4 实战：从Web布局到Avalonia布局

让我们通过一个实际案例，将一个Web页面转换为Avalonia布局。

### 3.4.1 Web页面示例

**登录表单（Vue实现）：**

```vue
<template>
  <div class="login-container">
    <div class="login-box">
      <h1 class="title">用户登录</h1>

      <form class="form" @submit.prevent="handleLogin">
        <div class="form-group">
          <label>用户名</label>
          <input v-model="username"
                 type="text"
                 placeholder="请输入用户名" />
        </div>

        <div class="form-group">
          <label>密码</label>
          <input v-model="password"
                 type="password"
                 placeholder="请输入密码" />
        </div>

        <div class="form-group">
          <label>
            <input v-model="rememberMe" type="checkbox" />
            记住我
          </label>
        </div>

        <button type="submit" class="login-btn">
          登录
        </button>

        <p class="footer-text">
          还没有账号？<a href="/register">立即注册</a>
        </p>
      </form>

  </div>
</template>

  console.log('登录：', username.value)
```

### 3.4.2 转换为Avalonia

**MainWindow.axaml：**

```xml
<Window xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:vm="using:MyApp.ViewModels"
        xmlns:d="http://schemas.microsoft.com/expression/blend/2008"
        xmlns:mc="http://schemas.openxmlformats.org/markup-compatibility/2006"
        mc:Ignorable="d" d:DesignWidth="800" d:DesignHeight="450"
        x:Class="MyApp.Views.MainWindow"
        x:DataType="vm:MainViewModel"
        Title="用户登录"
        Width="500"
        Height="600"
        WindowStartupLocation="CenterScreen">

  <Window.Styles>
    <!-- 定义样式 -->
    <Style Selector="TextBlock.title">
      <Setter Property="FontSize" Value="28"/>
      <Setter Property="FontWeight" Value="Bold"/>
      <Setter Property="HorizontalAlignment" Value="Center"/>
      <Setter Property="Margin" Value="0,0,0,30"/>
      <Setter Property="Foreground" Value="#333333"/>
    </Style>

    <Style Selector="TextBox.form-input">
      <Setter Property="Width" Value="320"/>
      <Setter Property="Height" Value="40"/>
      <Setter Property="Padding" Value="12,0"/>
      <Setter Property="BorderBrush" Value="#dddddd"/>
      <Setter Property="BorderThickness" Value="1"/>
      <Setter Property="CornerRadius" Value="4"/>
    </Style>

    <Style Selector="Button.login-btn">
      <Setter Property="Width" Value="320"/>
      <Setter Property="Height" Value="44"/>
      <Setter Property="Background" Value="#667eea"/>
      <Setter Property="Foreground" Value="White"/>
      <Setter Property="BorderThickness" Value="0"/>
      <Setter Property="CornerRadius" Value="4"/>
      <Setter Property="FontSize" Value="16"/>
      <Setter Property="Cursor" Value="Hand"/>
    </Style>

    <Style Selector="Button.login-btn:pointerover">
      <Setter Property="Background" Value="#5568d3"/>
    </Style>
  </Window.Styles>

  <!-- 主容器 -->
  <Grid>
    <!-- 背景渐变 -->
    <Grid.Background>
      <LinearGradientBrush StartPoint="0%,0%" EndPoint="100%,100%">
        <GradientStop Color="#667eea" Offset="0"/>
        <GradientStop Color="#764ba2" Offset="1"/>
      </LinearGradientBrush>
    </Grid.Background>

    <!-- 登录框（居中） -->
    <Border Width="400"
            Padding="40"
            Background="White"
            CornerRadius="8"
            BoxShadow="0 4 6 0 rgba(0,0,0,0.1)"
            HorizontalAlignment="Center"
            VerticalAlignment="Center">

      <StackPanel Spacing="20">
        <!-- 标题 -->
        <TextBlock Text="用户登录"
                   Classes="title"/>

        <!-- 用户名 -->
        <StackPanel Spacing="8">
          <TextBlock Text="用户名"
                     Foreground="#666666"/>
          <TextBox Text="{Binding Username}"
                   PlaceholderText="请输入用户名"
                   Classes="form-input"/>
        </StackPanel>

        <!-- 密码 -->
        <StackPanel Spacing="8">
          <TextBlock Text="密码"
                     Foreground="#666666"/>
          <TextBox Text="{Binding Password}"
                   PasswordChar="•"
                   PlaceholderText="请输入密码"
                   Classes="form-input"/>
        </StackPanel>

        <!-- 记住我 -->
        <CheckBox Content="记住我"
                  IsChecked="{Binding RememberMe}"/>

        <!-- 登录按钮 -->
        <Button Content="登录"
                Classes="login-btn"
                Command="{Binding LoginCommand}"/>

        <!-- 底部文本 -->
        <StackPanel Orientation="Horizontal"
                    HorizontalAlignment="Center"
                    Spacing="4">
          <TextBlock Text="还没有账号？"
                     Foreground="#999999"/>
          <HyperlinkButton Content="立即注册"
                           Foreground="#667eea"
                           Command="{Binding GoToRegisterCommand}"/>
        </StackPanel>
      </StackPanel>

    </Border>
  </Grid>

</Window>
```

### 3.4.3 转换要点总结

1. **容器映射**：
   - `<div class="login-container">` → `<Grid>`（带背景）
   - `<div class="login-box">` → `<Border>`（带阴影和圆角）

2. **布局映射**：
   - Flexbox居中 → `HorizontalAlignment="Center"` + `VerticalAlignment="Center"`
   - `<form>` → `<StackPanel>`

3. **控件映射**：
   - `<input type="text">` → `<TextBox>`
   - `<input type="password">` → `<TextBox PasswordChar="•">`
   - `<input type="checkbox">` → `<CheckBox>`
   - `<button>` → `<Button>`
   - `<a>` → `<HyperlinkButton>`

4. **样式映射**：
   - CSS类 → Avalonia Styles
   - `:hover`伪类 → `:pointerover`选择器

5. **数据绑定**：
   - `v-model` → `{Binding Property, Mode=TwoWay}`
   - `@submit` → `Command`
