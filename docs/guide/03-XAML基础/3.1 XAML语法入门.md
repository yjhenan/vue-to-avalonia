## 3.1 XAML语法入门

### 3.1.1 元素和属性

**HTML基本结构：**

```html
<!-- HTML -->
<div class="container">
  <h1 id="title">Hello World</h1>
  <input type="text" placeholder="请输入" disabled />
</div>
```

**XAML基本结构：**

```xml
<!-- XAML -->
<StackPanel Classes="container">
  <TextBlock x:Name="Title"
             Text="Hello World"
             FontSize="24"/>
  <TextBox PlaceholderText="请输入"
           IsEnabled="False"/>
</StackPanel>
```

**关键区别对照：**

| HTML概念 | XAML概念 | 说明 |
|---------|---------|------|
| 标签（Tag） | 元素（Element） | 都用尖括号表示 |
| `class`属性 | `Classes`属性 | 样式类名 |
| `id`属性 | `x:Name`属性 | 元素标识符 |
| 属性值 | 属性值 | XAML中必须用引号 |
| 自闭合标签 | 自闭合标签 | `<Tag />`形式 |
| 嵌套关系 | 嵌套关系 | 父子元素关系 |

### 3.1.2 命名空间

**HTML只有一个默认命名空间：**

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
</head>
<body>
  <!-- 元素默认在HTML命名空间 -->
</body>
</html>
```

**XAML需要显式声明命名空间：**

```xml
<Window xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:vm="clr-namespace:MyApp.ViewModels"
        xmlns:local="clr-namespace:MyApp"
        xmlns:d="http://schemas.microsoft.com/expression/blend/2008"
        xmlns:mc="http://schemas.openxmlformats.org/markup-compatibility/2006"
        mc:Ignorable="d"
        x:Class="MyApp.MainWindow">

  <!-- Window内容 -->

</Window>
```

**命名空间解析：**

| 命名空间声明 | 用途 | 对比HTML |
|------------|------|---------|
| `xmlns="https://github.com/avaloniaui"` | Avalonia默认命名空间（包含所有内置控件） | 类似HTML默认命名空间 |
| `xmlns:x="..."` | XAML语言特性（x:Name, x:Class等） | 无对应 |
| `xmlns:vm="clr-namespace:MyApp.ViewModels"` | 引用项目中的命名空间 | 类似import |
| `xmlns:local="clr-namespace:MyApp"` | 引用当前项目命名空间 | 类似相对导入 |
| `xmlns:d="..."` | 设计时特性（设计预览） | 无对应 |
| `mc:Ignorable="d"` | 忽略设计时标记 | 无对应 |

**自定义命名空间前缀：**

```xml
<!-- 引用第三方库 -->
xmlns:material="clr-namespace:Material.Avalonia;assembly=Material.Avalonia"

<!-- 使用 -->
<material:Card>
  <TextBlock Text="Material Design Card"/>
</material:Card>
```

### 3.1.3 属性语法

XAML有两种设置属性的方式：**Attribute语法**和**Property元素语法**。

**Attribute语法（类似HTML属性）：**

```xml
<!-- 简单属性 -->
<Button Content="点击我"
        Width="100"
        Height="30"
        Background="Blue"
        Foreground="White"/>
```

**Property元素语法（复杂属性）：**

```xml
<!-- 当属性值很复杂时，可以使用元素语法 -->
<Button Width="100"
        Height="30">
  <Button.Content>
    <StackPanel>
      <TextBlock Text="图标"/>
      <TextBlock Text="点击我"/>
    </StackPanel>
  </Button.Content>
  <Button.Background>
    <LinearGradientBrush StartPoint="0%,0%" EndPoint="100%,100%">
      <GradientStop Color="Blue" Offset="0"/>
      <GradientStop Color="Purple" Offset="1"/>
    </LinearGradientBrush>
  </Button.Background>
</Button>
```

**HTML无法实现复杂属性，需要用嵌套元素：**

```html
<!-- HTML实现类似效果 -->
<button style="width: 100px; height: 30px;">
  <span class="icon">图标</span>
  <span>点击我</span>
</button>
```

**附加属性（Attached Properties）：**

这是XAML特有的概念，允许子元素访问父容器的特殊属性：

```xml
<Grid>
  <!-- Grid.Row和Grid.Column是附加属性 -->
  <TextBlock Text="第0行第0列"
             Grid.Row="0"
             Grid.Column="0"/>
  <TextBlock Text="第1行第2列"
             Grid.Row="1"
             Grid.Column="2"/>
</Grid>
```

### 3.1.4 内容属性（Content Property）

很多控件有一个"内容属性"，可以直接写在元素内部，无需显式标注：

```xml
<!-- 完整写法 -->
<Button>
  <Button.Content>
    <TextBlock Text="点击我"/>
  </Button.Content>
</Button>

<!-- 简写（Content是Button的内容属性） -->
<Button>
  <TextBlock Text="点击我"/>
</Button>

<!-- 更简单的写法（字符串直接作为Content） -->
<Button Content="点击我"/>
```

**Vue对比：**

```vue
<!-- Vue的slot默认内容 -->
<button>
  <span>点击我</span>
</button>
```
