## 3.3 基础控件映射表

### 3.3.1 常用控件对照

| HTML元素 | Avalonia控件 | 说明 |
|---------|------------|------|
| `<div>` | `Panel`, `Border`, `ContentControl` | 容器元素 |
| `<span>` | `TextBlock`, `Run` | 行内文本 |
| `<p>` | `TextBlock` | 段落文本 |
| `<h1>-<h6>` | `TextBlock`（设置FontSize） | 标题 |
| `<button>` | `Button` | 按钮 |
| `<input type="text">` | `TextBox` | 文本输入框 |
| `<input type="password">` | `TextBox`（PasswordChar="•"） | 密码输入 |
| `<input type="checkbox">` | `CheckBox` | 复选框 |
| `<input type="radio">` | `RadioButton` | 单选按钮 |
| `<select>` | `ComboBox` | 下拉选择 |
| `<textarea>` | `TextBox`（AcceptsReturn="True"） | 多行文本 |
| `<img>` | `Image` | 图片 |
| `<a>` | `HyperlinkButton` | 超链接 |
| `<ul>/<ol>` | `ItemsControl` | 列表 |
| `<table>` | `DataGrid` | 表格 |
| `<video>` | `MediaPlayer`（需要第三方库） | 视频 |
| `<audio>` | `MediaPlayer`（需要第三方库） | 音频 |

### 3.3.2 常用属性对照

**文本相关：**

```xml
<!-- TextBlock（类似span/p） -->
<TextBlock Text="Hello World"
           FontSize="24"
           FontWeight="Bold"
           FontStyle="Italic"
           Foreground="Blue"
           TextAlignment="Center"
           TextWrapping="Wrap"
           MaxLines="2"/>

<!-- Vue对比 -->
<!-- <p style="font-size: 24px; font-weight: bold; font-style: italic;
            color: blue; text-align: center; max-lines: 2;">
       Hello World
     </p> -->
```

**按钮相关：**

```xml
<!-- Button -->
<Button Content="点击我"
        Width="120"
        Height="40"
        IsEnabled="True"
        IsVisible="True"
        Cursor="Hand"
        Click="Button_Click"/>

<!-- Vue对比 -->
<!-- <button style="width: 120px; height: 40px;"
           :disabled="!isEnabled"
           v-show="isVisible"
           @click="handleClick">
       点击我
     </button> -->
```

**输入框相关：**

```xml
<!-- TextBox -->
<TextBox Text="{Binding Username, Mode=TwoWay}"
         PlaceholderText="请输入用户名"
         Width="200"
         MaxLength="50"
         IsReadOnly="False"
         PasswordChar="•"/>

<!-- Vue对比 -->
<!-- <input v-model="username"
          placeholder="请输入用户名"
          style="width: 200px;"
          maxlength="50"
          :readonly="isReadOnly"
          type="password" /> -->
```

### 3.3.3 控件属性对比（v-bind vs Binding）

**Vue的v-bind：**

```vue
<template>
  <div>
    <!-- 静态绑定 -->
    <button :disabled="isDisabled">按钮</button>

    <!-- 动态class绑定 -->
    <div :class="{ active: isActive, error: hasError }">内容</div>

    <!-- 动态style绑定 -->
    <p :style="{ color: textColor, fontSize: fontSize + 'px' }">
      文本
    </p>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const isDisabled = ref(false)
const isActive = ref(true)
const hasError = ref(false)
const textColor = ref('red')
const fontSize = ref(16)
</script>
```

**Avalonia的Binding：**

```xml
<StackPanel xmlns="https://github.com/avaloniaui">

  <!-- 绑定到ViewModel属性 -->
  <Button IsEnabled="{Binding !IsDisabled}"
          Content="按钮"/>

  <!-- 类绑定 -->
  <Border Classes="{Binding ActiveClass}"
          Background="{Binding BackgroundColor}">
    <TextBlock Text="内容"/>
  </Border>

  <!-- 字符串格式化 -->
  <TextBlock Text="{Binding Price, StringFormat='{}{0:C}'}"/>

</StackPanel>
```

```csharp
// ViewModel
using CommunityToolkit.Mvvm.ComponentModel;

public partial class MainViewModel : ObservableObject
{
    [ObservableProperty] private bool isDisabled = false;
    [ObservableProperty] private string activeClass = "active";
    [ObservableProperty] private string backgroundColor = "LightBlue";
}
```

**关键区别：**

| 特性 | Vue (v-bind) | Avalonia (Binding) |
|-----|-------------|-------------------|
| **语法** | `:prop="value"` | `{Binding Path}` |
| **双向绑定** | `v-model` | `Mode=TwoWay` |
| **表达式** | 可以写JavaScript表达式 | 只能绑定属性路径 |
| **转换器** | computed或方法 | Converter类 |
| **调试** | Vue DevTools | Avalonia DevTools |
