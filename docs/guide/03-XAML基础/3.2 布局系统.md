## 3.2 布局系统

Avalonia的布局系统与CSS布局有很大不同，但它更加明确和强大。

### 3.2.1 常用布局控件

#### Grid（网格布局）

**Grid是最强大的布局控件，类似CSS Grid但更简单：**

```xml
<!-- Grid布局 -->
<Grid>
  <!-- 定义行 -->
  <Grid.RowDefinitions>
    <RowDefinition Height="Auto"/>   <!-- 自动高度 -->
    <RowDefinition Height="*"/>      <!-- 占据剩余空间 -->
    <RowDefinition Height="100"/>    <!-- 固定高度 -->
  </Grid.RowDefinitions>

  <!-- 定义列 -->
  <Grid.ColumnDefinitions>
    <ColumnDefinition Width="200"/>  <!-- 固定宽度 -->
    <ColumnDefinition Width="*"/>    <!-- 占据剩余空间 -->
    <ColumnDefinition Width="Auto"/> <!-- 自动宽度 -->
  </Grid.ColumnDefinitions>

  <!-- 放置元素 -->
  <TextBlock Text="Header"
             Grid.Row="0"
             Grid.Column="0"
             Grid.ColumnSpan="3"/> <!-- 跨3列 -->

  <TextBox Text="Content"
           Grid.Row="1"
           Grid.Column="1"/>
</Grid>
```

**CSS Grid对比：**

```html
<!-- CSS Grid实现 -->
<div style="display: grid;
            grid-template-rows: auto 1fr 100px;
            grid-template-columns: 200px 1fr auto;">
  <div style="grid-row: 0; grid-column: 0 / span 3;">
    Header
  </div>
  <div style="grid-row: 1; grid-column: 1;">
    Content
  </div>
</div>
```

**Grid尺寸单位：**

| XAML单位 | 说明 | CSS对应 |
|---------|------|---------|
| `Auto` | 自动大小 | `auto` |
| `*` | 比例大小 | `1fr` |
| `2*` | 2倍比例 | `2fr` |
| `100` | 固定像素 | `100px` |

#### StackPanel（堆叠面板）

**StackPanel让元素水平或垂直堆叠：**

```xml
<!-- 垂直堆叠（默认） -->
<StackPanel Orientation="Vertical"
            Spacing="10">  <!-- 间距 -->
  <Button Content="按钮1"/>
  <Button Content="按钮2"/>
  <Button Content="按钮3"/>
</StackPanel>

<!-- 水平堆叠 -->
<StackPanel Orientation="Horizontal"
            Spacing="10">
  <Button Content="左"/>
  <Button Content="中"/>
  <Button Content="右"/>
</StackPanel>
```

**CSS Flexbox对比：**

```html
<!-- 垂直堆叠 -->
<div style="display: flex;
            flex-direction: column;
            gap: 10px;">
  <button>按钮1</button>
  <button>按钮2</button>
  <button>按钮3</button>
</div>

<!-- 水平堆叠 -->
<div style="display: flex; gap: 10px;">
  <button>左</button>
  <button>中</button>
  <button>右</button>
</div>
```

#### DockPanel（停靠面板）

**DockPanel让元素停靠在四周：**

```xml
<DockPanel LastChildFill="True">
  <!-- 停靠在顶部 -->
  <TextBlock Text="Header"
             DockPanel.Dock="Top"
             Background="LightBlue"/>

  <!-- 停靠在底部 -->
  <TextBlock Text="Footer"
             DockPanel.Dock="Bottom"
             Background="LightGray"/>

  <!-- 停靠在左侧 -->
  <TextBlock Text="Sidebar"
             DockPanel.Dock="Left"
             Background="LightYellow"/>

  <!-- 中间内容（自动填充） -->
  <TextBlock Text="Content"
             Background="White"/>
</DockPanel>
```

**无直接CSS对应，需要组合定位：**

```html
<div style="display: flex; flex-direction: column; height: 100vh;">
  <header style="height: 60px;">Header</header>
  <div style="display: flex; flex: 1;">
    <aside style="width: 200px;">Sidebar</aside>
    <main style="flex: 1;">Content</main>
  </div>
  <footer style="height: 40px;">Footer</footer>
</div>
```

#### WrapPanel（换行面板）

**WrapPanel让元素自动换行：**

```xml
<WrapPanel Orientation="Horizontal"
           ItemWidth="100"
           ItemHeight="100">
  <Border Background="Red"/>
  <Border Background="Green"/>
  <Border Background="Blue"/>
  <Border Background="Yellow"/>
  <!-- 放不下会自动换行 -->
</WrapPanel>
```

**CSS Flexbox对比：**

```html
<div style="display: flex;
            flex-wrap: wrap;
            gap: 10px;">
  <div style="width: 100px; height: 100px; background: red;"></div>
  <div style="width: 100px; height: 100px; background: green;"></div>
  <div style="width: 100px; height: 100px; background: blue;"></div>
  <!-- 自动换行 -->
</div>
```

#### Canvas（绝对定位）

**Canvas使用绝对坐标定位：**

```xml
<Canvas Width="400" Height="300">
  <Button Content="按钮1"
          Canvas.Left="50"
          Canvas.Top="100"/>

  <Button Content="按钮2"
          Canvas.Left="150"
          Canvas.Top="200"/>

  <Ellipse Width="50"
           Height="50"
           Fill="Red"
           Canvas.Left="200"
           Canvas.Top="50"/>
</Canvas>
```

**CSS绝对定位对比：**

```html
<div style="position: relative; width: 400px; height: 300px;">
  <button style="position: absolute; left: 50px; top: 100px;">按钮1</button>
  <button style="position: absolute; left: 150px; top: 200px;">按钮2</button>
  <div style="position: absolute; left: 200px; top: 50px;
              width: 50px; height: 50px; background: red;
              border-radius: 50%;"></div>
</div>
```

### 3.2.2 布局对比实战

**实战案例：经典三栏布局**

**Vue + CSS实现：**

```vue
<template>
  <div class="layout">
    <header class="header">Header</header>
    <div class="main">
      <aside class="sidebar">Sidebar</aside>
      <main class="content">Content</main>
    </div>
    <footer class="footer">Footer</footer>
  </div>
</template>

<style scoped>
.layout {
  display: flex;
  flex-direction: column;
  height: 100vh;
}

.header {
  height: 60px;
  background: #4a90e2;
}

.main {
  display: flex;
  flex: 1;
}

.sidebar {
  width: 200px;
  background: #f0f0f0;
}

.content {
  flex: 1;
  background: white;
}

.footer {
  height: 40px;
  background: #e0e0e0;
}
</style>
```

**Avalonia XAML实现：**

```xml
<Grid xmlns="https://github.com/avaloniaui">
  <Grid.RowDefinitions>
    <RowDefinition Height="60"/>    <!-- Header固定高度 -->
    <RowDefinition Height="*"/>     <!-- Main占据剩余 -->
    <RowDefinition Height="40"/>    <!-- Footer固定高度 -->
  </Grid.RowDefinitions>

  <!-- Header -->
  <Border Grid.Row="0"
          Background="#4a90e2">
    <TextBlock Text="Header"
               HorizontalAlignment="Center"
               VerticalAlignment="Center"/>
  </Border>

  <!-- Main -->
  <Grid Grid.Row="1">
    <Grid.ColumnDefinitions>
      <ColumnDefinition Width="200"/> <!-- Sidebar固定宽度 -->
      <ColumnDefinition Width="*"/>   <!-- Content占据剩余 -->
    </Grid.ColumnDefinitions>

    <!-- Sidebar -->
    <Border Grid.Column="0"
            Background="#f0f0f0">
      <TextBlock Text="Sidebar"
                 HorizontalAlignment="Center"
                 VerticalAlignment="Center"/>
    </Border>

    <!-- Content -->
    <Border Grid.Column="1"
            Background="White">
      <TextBlock Text="Content"
                 HorizontalAlignment="Center"
                 VerticalAlignment="Center"/>
    </Border>
  </Grid>

  <!-- Footer -->
  <Border Grid.Row="2"
          Background="#e0e0e0">
    <TextBlock Text="Footer"
               HorizontalAlignment="Center"
               VerticalAlignment="Center"/>
  </Border>

</Grid>
```

**对比总结：**

| 特性 | CSS布局 | Avalonia布局 |
|-----|---------|-------------|
| **学习曲线** | 简单到复杂（Flexbox → Grid） | 中等（需要理解面板概念） |
| **表达能力** | 非常灵活 | 非常明确 |
| **调试** | 浏览器DevTools | Avalonia DevTools |
| **响应式** | 媒体查询 | 需要代码支持 |
| **性能** | 依赖浏览器 | 原生优化 |

### 3.2.3 对齐和边距

**对齐属性：**

```xml
<StackPanel>
  <!-- 水平对齐 -->
  <Button Content="左对齐"
          HorizontalAlignment="Left"
          Width="100"/>

  <Button Content="居中"
          HorizontalAlignment="Center"
          Width="100"/>

  <Button Content="右对齐"
          HorizontalAlignment="Right"
          Width="100"/>

  <Button Content="拉伸填充"
          HorizontalAlignment="Stretch"/>

  <!-- 垂直对齐 -->
  <Button Content="顶部"
          VerticalAlignment="Top"
          Height="50"/>

  <Button Content="居中"
          VerticalAlignment="Center"
          Height="50"/>

  <Button Content="底部"
          VerticalAlignment="Bottom"
          Height="50"/>
</StackPanel>
```

**CSS对比：**

```css
/* 水平对齐 */
.left { align-self: flex-start; }
.center { align-self: center; }
.right { align-self: flex-end; }
.stretch { align-self: stretch; }

/* 垂直对齐 */
.top { justify-self: start; }
.center { justify-self: center; }
.bottom { justify-self: end; }
```

**边距（Margin）和内边距（Padding）：**

```xml
<!-- Margin：外边距（元素外部空间） -->
<!-- Padding：内边距（元素内部空间） -->
<Border Background="LightBlue"
        Padding="20"          <!-- 四周都是20 -->
        Margin="10,20,10,20"> <!-- 左,上,右,下 -->

  <Button Content="按钮"
          Margin="5,10"/>     <!-- 水平5，垂直10 -->

</Border>

<!-- 简写形式 -->
<!-- Margin="10" → 四周都是10 -->
<!-- Margin="10,20" → 水平10，垂直20 -->
<!-- Margin="10,20,30,40" → 左,上,右,下 -->
```

**CSS对比：**

```css
.border {
  padding: 20px;
  margin: 10px 20px;
}

.button {
  margin: 10px 5px;
}
```
