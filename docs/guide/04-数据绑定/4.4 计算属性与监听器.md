## 4.4 计算属性与监听器

### 4.4.1 computed属性迁移

**Vue的computed：**

```vue
<template>
  <div>
    <p>全名：{{ fullName }}</p>
    <p>描述：{{ description }}</p>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')
const age = ref(25)

// 只读计算属性
const fullName = computed(() =>
  `${firstName.value} ${lastName.value}`
)

// 可写计算属性
const fullNameWritable = computed({
  get() {
    return `${firstName.value} ${lastName.value}`
  },
  set(newValue) {
    [firstName.value, lastName.value] = newValue.split(' ')
  }
})

// 复杂计算属性
const description = computed(() => {
  const ageGroup = age.value >= 18 ? '成年人' : '未成年人'
  return `${fullName.value}，${ageGroup}`
})
</script>
```

**C#属性实现：**

```csharp
using CommunityToolkit.Mvvm.ComponentModel;

public partial class UserViewModel : ObservableObject
{
    [ObservableProperty]
    private string firstName = "John";

    [ObservableProperty]
    private string lastName = "Doe";

    [ObservableProperty]
    private int age = 25;

    // 依赖属性的更新通过 partial 方法自动触发
    partial void OnFirstNameChanged(string value) => OnPropertyChanged(nameof(FullName));
    partial void OnLastNameChanged(string value) => OnPropertyChanged(nameof(FullName));
    partial void OnAgeChanged(int value) => OnPropertyChanged(nameof(Description));

    // 只读计算属性（类似computed）
    public string FullName => $"{FirstName} {LastName}";

    // 复杂计算属性
    public string Description
    {
        get
        {
            string ageGroup = Age >= 18 ? "成年人" : "未成年人";
            return $"{FullName}，{ageGroup}";
        }
    }
}
```

**使用[ObservableProperty] + 计算属性：**

```csharp
using CommunityToolkit.Mvvm.ComponentModel;

public partial class UserViewModel : ObservableObject
{
    [ObservableProperty]
    private string firstName = "John";

    [ObservableProperty]
    private string lastName = "Doe";

    // 当FirstName或LastName改变时，通过partial方法通知FullName更新
    partial void OnFirstNameChanged(string value) => OnPropertyChanged(nameof(FullName));
    partial void OnLastNameChanged(string value) => OnPropertyChanged(nameof(FullName));

    // 只读计算属性（类似computed）
    public string FullName => $"{FirstName} {LastName}";
}
```

### 4.4.2 watch深度监听

**Vue的watch：**

```vue
<script setup>
import { ref, watch } from 'vue'

const count = ref(0)
const user = ref({
  name: 'Alice',
  preferences: {
    theme: 'dark'
  }
})

// 基本监听
watch(count, (newValue, oldValue) => {
  console.log(`count从${oldValue}变为${newValue}`)
})

// 深度监听对象
watch(user, (newValue) => {
  console.log('user改变了', newValue)
}, { deep: true })

// 监听特定属性
watch(() => user.value.preferences.theme, (newTheme) => {
  console.log('主题改变了', newTheme)
})

// 立即执行
watch(count, (newValue) => {
  console.log('立即执行一次', newValue)
}, { immediate: true })
</script>
```

**CommunityToolkit.Mvvm的OnChanged：**

```csharp
using CommunityToolkit.Mvvm.ComponentModel;

public partial class UserViewModel : ObservableObject
{
    [ObservableProperty]
    private int count = 0;

    [ObservableProperty]
    private User user = new User();

    // 基本监听（类似watch）—— 当Count变化时自动调用
    partial void OnCountChanged(int value)
    {
        Console.WriteLine($"Count变为{value}");
    }

    // 监听对象变化
    partial void OnUserChanged(User value)
    {
        Console.WriteLine($"用户改变了：{value.Name}");
    }
}

public class User
{
    public string Name { get; set; } = "Alice";
    public Preferences Preferences { get; set; } = new Preferences();
}

public class Preferences
{
    public string Theme { get; set; } = "dark";
}
```

**对比Vue的deep/immediate选项：**

```csharp
// Vue: { immediate: true }
// C#: OnChanged partial方法在属性首次设置时即会调用
// [ObservableProperty] 生成的 OnXxxChanged 会在每次属性变更时调用
// （包括首次赋值，因此天然等价于 immediate: true）
```
