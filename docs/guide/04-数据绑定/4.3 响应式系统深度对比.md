## 4.3 响应式系统深度对比

### 4.3.1 Vue响应式原理回顾

**Vue 3的响应式系统（基于Proxy）：**

```javascript
// Vue 3响应式原理简化版
const target = {
  message: 'Hello',
  count: 0
}

// Proxy拦截
const proxy = new Proxy(target, {
  get(target, key, receiver) {
    // 依赖收集
    track(target, key)
    return Reflect.get(target, key, receiver)
  },

  set(target, key, value, receiver) {
    const result = Reflect.set(target, key, value, receiver)
    // 触发更新
    trigger(target, key)
    return result
  }
})

// 依赖收集
let activeEffect = null
const targetMap = new WeakMap()

function track(target, key) {
  if (!activeEffect) return

  let depsMap = targetMap.get(target)
  if (!depsMap) {
    targetMap.set(target, (depsMap = new Map()))
  }

  let dep = depsMap.get(key)
  if (!dep) {
    depsMap.set(key, (dep = new Set()))
  }

  dep.add(activeEffect)
}

// 触发更新
function trigger(target, key) {
  const depsMap = targetMap.get(target)
  if (!depsMap) return

  const dep = depsMap.get(key)
  if (dep) {
    dep.forEach(effect => effect())
  }
}
```

**Vue实际使用：**

```vue
<script setup>
import { ref, reactive } from 'vue'

// ref用于基本类型
const count = ref(0)

// reactive用于对象
const state = reactive({
  message: 'Hello',
  user: {
    name: 'Alice',
    age: 25
  }
})

// 修改数据 → 自动触发UI更新
count.value++
state.message = 'Hi'
state.user.age = 26

// Vue自动追踪依赖
// 在模板中使用：{{ count }} → 自动建立依赖关系
</script>
```

**Vue响应式的限制：**

```javascript
// 1. 对象新增属性不会触发更新
const obj = reactive({ a: 1 })
obj.b = 2  // ❌ 不会触发更新（Vue 3已经解决，但Vue 2有此问题）

// 2. 数组下标直接赋值
const arr = reactive([1, 2, 3])
arr[0] = 10  // Vue 3可以，Vue 2不行

// 3. 解构会失去响应性
const { message } = state  // ❌ message不是响应式的
// 需要：
const { message } = toRefs(state)  // ✅
```

### 4.3.2 Avalonia响应式机制

**基于INotifyPropertyChanged接口：**

```csharp
// INotifyPropertyChanged定义
public interface INotifyPropertyChanged
{
    event PropertyChangedEventHandler PropertyChanged;
}

// 最基础的实现
public class UserViewModel : INotifyPropertyChanged
{
    private string name = string.Empty;

    public event PropertyChangedEventHandler? PropertyChanged;

    public string Name
    {
        get => name;
        set
        {
            if (name != value)
            {
                name = value;
                OnPropertyChanged(nameof(Name));
            }
        }
    }

    protected virtual void OnPropertyChanged(string propertyName)
    {
        PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
    }
}
```

**CommunityToolkit.Mvvm简化实现：**

```csharp
using CommunityToolkit.Mvvm.ComponentModel;

// CommunityToolkit.Mvvm 使用源生成器简化实现
public partial class UserViewModel : ObservableObject
{
    [ObservableProperty]
    private string name = string.Empty;

    [ObservableProperty]
    private int age = 0;
}

// 使用
var viewModel = new UserViewModel();
viewModel.Name = "Alice";  // 自动触发PropertyChanged事件
```

**PropertyChanged事件触发模式对比：**

| 特性 | Vue Proxy | INotifyPropertyChanged |
|-----|-----------|----------------------|
| **自动化程度** | 全自动 | 需要手动触发 |
| **类型安全** | 运行时检查 | 编译时检查 |
| **性能** | 中等（Proxy开销） | 高（直接赋值） |
| **调试** | 较难（Proxy黑盒） | 容易（显式触发） |
| **IDE支持** | 一般 | 强大的智能提示 |

### 4.3.3 集合响应式对比

**Vue的数组响应式：**

```vue
<script setup>
import { ref } from 'vue'

const todos = ref([
  { id: 1, title: '学习Vue', completed: false },
  { id: 2, title: '学习Avalonia', completed: false }

// ✅ Vue 3支持的数组操作

todos.value = newTodos
```

**Avalonia的ObservableCollection：**

```csharp
using System.Collections.ObjectModel;
using System.Collections.Specialized;

// ObservableCollection自动实现集合变化通知
public partial class TodoViewModel : ObservableObject
{
    public ObservableCollection<TodoItem> Todos { get; }

    public TodoViewModel()
    {
        Todos = new ObservableCollection<TodoItem>
        {
            new TodoItem { Id = 1, Title = "学习Vue", Completed = false },
            new TodoItem { Id = 2, Title = "学习Avalonia", Completed = false }
        };

        // 监听集合变化
        Todos.CollectionChanged += OnTodosCollectionChanged;
    }

    private void OnTodosCollectionChanged(object? sender, NotifyCollectionChangedEventArgs e)
    {
        if (e.Action == NotifyCollectionChangedAction.Add)
        {
            Console.WriteLine($"添加了{e.NewItems?.Count}个元素");
        }
        else if (e.Action == NotifyCollectionChangedAction.Remove)
        {
            Console.WriteLine($"删除了{e.OldItems?.Count}个元素");
        }
    }

    // 支持的操作
    public void AddTodo(TodoItem item)
    {
        Todos.Add(item);  // ✅ 触发CollectionChanged
    }

    public void RemoveTodo(TodoItem item)
    {
        Todos.Remove(item);  // ✅ 触发CollectionChanged
    }

    public void RemoveAt(int index)
    {
        Todos.RemoveAt(index);  // ✅ 触发CollectionChanged
    }

    public void Insert(int index, TodoItem item)
    {
        Todos.Insert(index, item);  // ✅ 触发CollectionChanged
    }

    public void Clear()
    {
        Todos.Clear();  // ✅ 触发CollectionChanged
    }
}
```

**⚠️ ObservableCollection的限制：**

```csharp
// ❌ 直接修改元素的属性不会触发UI更新
Todos[0].Title = "修改后的标题";  // 集合没变化，只是元素属性变了

// ✅ 解决方案1：元素也实现INotifyPropertyChanged
public partial class TodoItem : ObservableObject
{
    [ObservableProperty]
    private string title = string.Empty;
}

// ✅ 解决方案2：移除再添加
var item = Todos[0];
Todos.Remove(item);
item.Title = "修改后的标题";
Todos.Insert(0, item);

// ✅ 解决方案3：手动触发重置
Todos.Reset();
```

**性能对比：**

| 操作 | Vue Array | ObservableCollection |
|-----|-----------|---------------------|
| **添加元素** | O(1) | O(1) + 事件触发 |
| **删除元素** | O(n) | O(n) + 事件触发 |
| **批量操作** | 一次触发 | 多次触发（可优化） |
| **大数据量** | 虚拟滚动 | UI虚拟化 |

**批量更新优化：**

```csharp
// 批量添加时避免频繁触发事件
public void AddRange(IEnumerable<TodoItem> items)
{
    // 方案1：暂时取消订阅
    Todos.CollectionChanged -= OnTodosCollectionChanged;

    foreach (var item in items)
    {
        Todos.Add(item);
    }

    // 重新订阅
    Todos.CollectionChanged += OnTodosCollectionChanged;

    // 手动触发一次重置事件
    OnTodosCollectionChanged(Todos,
        new NotifyCollectionChangedEventArgs(NotifyCollectionChangedAction.Reset));
}

// 方案2：使用批量更新辅助方法
public void AddRange(IEnumerable<TodoItem> items)
{
    foreach (var item in items)
    {
        Todos.Add(item);
    }
    // 手动触发一次重置事件
    OnTodosCollectionChanged(Todos,
        new NotifyCollectionChangedEventArgs(NotifyCollectionChangedAction.Reset));
}
```

**响应式链式调用：**

```csharp
using CommunityToolkit.Mvvm.ComponentModel;

public partial class AdvancedViewModel : ObservableObject
{
    [ObservableProperty]
    private string input = string.Empty;

    [ObservableProperty]
    private bool isValid = false;

    [ObservableProperty]
    private string validationMessage = string.Empty;

    // 响应式：输入变化 → 验证 → 更新状态
    partial void OnInputChanged(string value)
    {
        var result = ValidateInput(value);
        IsValid = result.IsValid;
        ValidationMessage = result.Message;
    }

    private (bool IsValid, string Message) ValidateInput(string input)
    {
        if (string.IsNullOrWhiteSpace(input))
            return (false, "不能为空");

        if (input.Length < 3)
            return (false, "长度至少3个字符");

        return (true, "验证通过");
    }
}
```
