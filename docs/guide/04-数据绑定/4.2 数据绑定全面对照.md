## 4.2 数据绑定全面对照

### 4.2.1 基础绑定语法对照表

**文本绑定对照**

| 功能 | Vue语法 | Avalonia语法 |
|-----|---------|------------|
| **文本插值** | `{{ message }}` | `<TextBlock Text="{Binding Message}"/>` |
| **HTML绑定** | `v-html="rawHtml"` | 无直接对应（用RichTextBox） |
| **属性绑定** | `v-bind:href="url"` | `<HyperlinkButton NavigateUri="{Binding Url}"/>` |
| **动态属性** | `:disabled="isDisabled"` | `IsEnabled="{Binding !IsDisabled}"` |

**详细示例对比：**

```vue
<!-- Vue文本绑定 -->
<template>
  <div>
    <!-- 文本插值 -->
    <p>{{ message }}</p>

    <!-- 表达式 -->
    <p>{{ message.toUpperCase() }}</p>

    <!-- 拼接 -->
    <p>{{ firstName + ' ' + lastName }}</p>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const message = ref('Hello Vue')
const firstName = ref('John')
const lastName = ref('Doe')
</script>
```

```xml
<!-- Avalonia文本绑定 -->
<StackPanel>
  <!-- 简单绑定 -->
  <TextBlock Text="{Binding Message}"/>

  <!-- 使用Converter转换（类似表达式） -->
  <TextBlock Text="{Binding Message, Converter={x:Static StringToUpperConverter.Instance}}"/>

  <!-- 多重绑定（类似拼接） -->
  <TextBlock>
    <TextBlock.Text>
      <MultiBinding StringFormat="{}{0} {1}">
        <Binding Path="FirstName"/>
        <Binding Path="LastName"/>
      </MultiBinding>
    </TextBlock.Text>
  </TextBlock>
</StackPanel>
```

```csharp
using CommunityToolkit.Mvvm.ComponentModel;

// ViewModel
public partial class MainViewModel : ObservableObject
{
    [ObservableProperty]
    private string message = "Hello Avalonia";

    [ObservableProperty]
    private string firstName = "John";

    [ObservableProperty]
    private string lastName = "Doe";

    // 或者提供计算属性
    public string FullName => $"{FirstName} {LastName}";
}
```

**属性绑定对照**

```vue
<!-- Vue属性绑定 -->
<template>
  <div>
    <!-- 静态属性 -->
    <img src="/image.png" />

    <!-- 动态属性 -->
    <img :src="imageUrl" :alt="imageAlt" />

    <!-- 布尔属性 -->
    <button :disabled="isLoading">提交</button>

    <!-- class绑定 -->
    <div :class="{ active: isActive, error: hasError }">内容</div>

    <!-- style绑定 -->
    <p :style="{ color: textColor, fontSize: fontSize + 'px' }">
      文本
    </p>
  </div>
</template>
```

```xml
<!-- Avalonia属性绑定 -->
<StackPanel>
  <!-- 静态属性 -->
  <Image Source="/Assets/image.png"/>

  <!-- 动态属性 -->
  <Image Source="{Binding ImageUrl}"/>

  <!-- 布尔属性（注意：IsEnabled是反向的） -->
  <Button IsEnabled="{Binding !IsLoading}"
          Content="提交"/>

  <!-- Classes绑定 -->
  <Border Classes="{Binding ContainerClasses}">
    <TextBlock Text="内容"/>
  </Border>

  <!-- 或者使用样式选择器 -->
  <Border Classes.active="{Binding IsActive}"
          Classes.error="{Binding HasError}">
    <TextBlock Text="内容"/>
  </Border>

  <!-- 样式绑定 -->
  <TextBlock Text="文本"
             Foreground="{Binding TextColor}"
             FontSize="{Binding FontSize}"/>
</StackPanel>
```

**双向绑定对照**

```vue
<!-- Vue双向绑定 -->
<template>
  <div>
    <!-- 文本输入 -->
    <input v-model="username" type="text" />

    <!-- 多行文本 -->
    <textarea v-model="description"></textarea>

    <!-- 复选框 -->
    <input v-model="agree" type="checkbox" />

    <!-- 单选框 -->
    <input v-model="gender" type="radio" value="male" />
    <input v-model="gender" type="radio" value="female" />

    <!-- 下拉选择 -->
    <select v-model="selectedCity">
      <option value="beijing">北京</option>
      <option value="shanghai">上海</option>
    </select>
  </div>
</template>
```

```xml
<!-- Avalonia双向绑定 -->
<StackPanel Spacing="10">
  <!-- 文本输入 -->
  <TextBox Text="{Binding Username, Mode=TwoWay}"/>

  <!-- 多行文本 -->
  <TextBox Text="{Binding Description, Mode=TwoWay}"
           AcceptsReturn="True"
           Height="100"/>

  <!-- 复选框 -->
  <CheckBox Content="同意条款"
            IsChecked="{Binding Agree, Mode=TwoWay}"/>

  <!-- 单选框 -->
  <StackPanel>
    <RadioButton Content="男"
                 IsChecked="{Binding Gender, Converter={x:Static MaleConverter.Instance}, Mode=TwoWay}"
                 GroupName="Gender"/>
    <RadioButton Content="女"
                 IsChecked="{Binding Gender, Converter={x:Static FemaleConverter.Instance}, Mode=TwoWay}"
                 GroupName="Gender"/>
  </StackPanel>

  <!-- 下拉选择 -->
  <ComboBox SelectedItem="{Binding SelectedCity, Mode=TwoWay}">
    <ComboBoxItem Content="北京" Tag="beijing"/>
    <ComboBoxItem Content="上海" Tag="shanghai"/>
  </ComboBox>
</StackPanel>
```

### 4.2.2 绑定模式详解

Avalonia提供了5种绑定模式，比Vue的模式更加明确：

**1. OneWay（默认）- 对比Vue单向数据流**

```xml
<!-- OneWay：源 → 目标 -->
<TextBlock Text="{Binding Message}"/>
<!-- 等价于 -->
<TextBlock Text="{Binding Message, Mode=OneWay}"/>
```

```csharp
// ViewModel
public string Message { get; set; } = "Hello";
// Message改变 → TextBlock自动更新
// 用户无法通过UI改变Message
```

**Vue对应：**

```vue
<!-- Vue中类似单向绑定 -->
<p>{{ message }}</p>
<!-- message改变 → 视图更新 -->
<!-- 但用户无法直接修改message -->
```

**2. TwoWay - 对比v-model**

```xml
<!-- TwoWay：源 ↔ 目标 -->
<TextBox Text="{Binding Username, Mode=TwoWay}"/>
```

```csharp
// ViewModel
[ObservableProperty]
private string username = string.Empty;

// Username改变 → TextBox更新
// 用户输入 → Username更新
```

**Vue对应：**

```vue
<input v-model="username" />
```

**3. OneTime - 性能优化场景**

```xml
<!-- OneTime：只在初始化时绑定一次 -->
<TextBlock Text="{Binding Title, Mode=OneTime}"/>
```

```csharp
// Title只在界面初始化时读取一次
// 之后Title改变不会更新UI
// 适用于不会变化的数据（如标题、标签）
public string Title { get; } = "设置";
```

**Vue没有直接对应，可以用 computed + 缓存：**

```javascript
const title = computed(() => '设置') // 永不改变
```

**4. OneWayToSource - Vue中没有的模式**

```xml
<!-- OneWayToSource：目标 → 源（反向绑定） -->
<Slider Value="{Binding ZoomLevel, Mode=OneWayToSource}"
        Minimum="0.5" Maximum="3.0"/>
```

```csharp
// 滑块改变 → ZoomLevel更新
// ZoomLevel改变 → 滑块不会更新（反向单向）
// 适用于只需要从UI读取数据，不需要回显的场景
```

**5. Default - 根据目标属性决定**

```xml
<!-- Default：使用目标属性的默认模式 -->
<TextBlock Text="{Binding Message, Mode=Default}"/>
<!-- TextBox.Text默认是TwoWay -->
<!-- TextBlock.Text默认是OneWay -->
```

**绑定模式对照表：**

| 绑定模式 | 方向 | Vue对应 | 使用场景 |
|---------|-----|---------|---------|
| OneWay | 源→目标 | `{{ }}`插值 | 显示只读数据 |
| TwoWay | 源↔目标 | `v-model` | 表单输入 |
| OneTime | 初始化一次 | 无 | 标题、标签等静态内容 |
| OneWayToSource | 目标→源 | 无 | 特殊场景（如配置项） |
| Default | 自动判断 | - | 让框架决定 |

### 4.2.3 高级绑定技巧

**绑定转换器（Converter）vs Vue filters/computed**

转换器用于在绑定过程中转换数据格式。

**Vue的filters和computed：**

```vue
<template>
  <div>
    <!-- 使用filter（Vue 2） -->
    <p>{{ price | currency }}</p>

    <!-- 使用computed（Vue 3） -->
    <p>{{ formattedPrice }}</p>

    <!-- 使用方法 -->
    <p>{{ formatDate(date) }}</p>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

const price = ref(99.99)

// computed方式
const formattedPrice = computed(() =>
  `¥${price.value.toFixed(2)}`
)

// 方法方式
function formatDate(date) {
  return new Date(date).toLocaleDateString('zh-CN')
}
</script>
```

**Avalonia的Converter：**

```csharp
// 定义Converter
using Avalonia.Data.Converters;
using System;
using System.Globalization;

public class CurrencyConverter : IValueConverter
{
    public static readonly CurrencyConverter Instance = new();

    public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
    {
        if (value is decimal price)
        {
            return $"¥{price:F2}";
        }
        return value;
    }

    public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)
    {
        // 双向绑定需要的反向转换
        if (value is string str && str.StartsWith("¥"))
        {
            if (decimal.TryParse(str.Substring(1), out var result))
                return result;
        }
        return value;
    }
}
```

```xml
<!-- 使用Converter -->
<Window xmlns:local="clr-namespace:MyApp.Converters">
  <Window.Resources>
    <local:CurrencyConverter x:Key="CurrencyConverter"/>
  </Window.Resources>

  <StackPanel>
    <!-- 使用Converter -->
    <TextBlock Text="{Binding Price, Converter={StaticResource CurrencyConverter}}"/>

    <!-- 带参数的Converter -->
    <TextBlock Text="{Binding Price, Converter={StaticResource CurrencyConverter}, ConverterParameter='USD'}"/>
  </StackPanel>
</Window>
```

**常用Converter示例：**

```csharp
// 1. 布尔取反转换器
public class BoolToInverseConverter : IValueConverter
{
    public static readonly BoolToInverseConverter Instance = new();

    public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
    {
        if (value is bool b)
            return !b;
        return value;
    }

    public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)
    {
        if (value is bool b)
            return !b;
        return value;
    }
}
```

```xml
<!-- 使用 -->
<Button IsEnabled="{Binding IsLoading, Converter={x:Static BoolToInverseConverter.Instance}}"/>
```

```csharp
// 2. 布尔到可见性转换器
public class BoolToVisibilityConverter : IValueConverter
{
    public static readonly BoolToVisibilityConverter Instance = new();

    public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
    {
        if (value is bool b)
            return b ? true : false; // 或者返回 Visibility.Visible / Collapsed
        return false;
    }

    public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)
    {
        throw new NotImplementedException();
    }
}
```

```xml
<!-- 使用 -->
<Border IsVisible="{Binding HasError, Converter={x:Static BoolToVisibilityConverter.Instance}}">
  <TextBlock Text="发生错误"/>
</Border>
```

```csharp
// 3. 枚举到布尔转换器（用于RadioButton）
public class EnumToBoolConverter : IValueConverter
{
    public static readonly EnumToBoolConverter Instance = new();

    public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
    {
        return value?.Equals(parameter) ?? false;
    }

    public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)
    {
        return (bool)value ? parameter : BindingOperations.DoNothing;
    }
}
```

```xml
<!-- 使用 -->
<RadioButton Content="男"
             IsChecked="{Binding Gender, Converter={x:Static EnumToBoolConverter.Instance}, ConverterParameter=Male, Mode=TwoWay}"
             GroupName="Gender"/>
<RadioButton Content="女"
             IsChecked="{Binding Gender, Converter={x:Static EnumToBoolConverter.Instance}, ConverterParameter=Female, Mode=TwoWay}"
             GroupName="Gender"/>
```

**多重绑定（MultiBinding）vs computed组合**

```vue
<!-- Vue computed组合 -->
<template>
  <p>{{ fullName }}</p>
</template>

<script setup>
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

const fullName = computed(() =>
  `${firstName.value} ${lastName.value}`
)
</script>
```

```xml
<!-- Avalonia MultiBinding -->
<TextBlock>
  <TextBlock.Text>
    <MultiBinding StringFormat="{}{0} {1}">
      <Binding Path="FirstName"/>
      <Binding Path="LastName"/>
    </MultiBinding>
  </TextBlock.Text>
</TextBlock>
```

**更复杂的MultiBinding（使用IMultiValueConverter）：**

```csharp
// 定义多值转换器
public class FullNameConverter : IMultiValueConverter
{
    public static readonly FullNameConverter Instance = new();

    public object Convert(IList<object> values, Type targetType, object parameter, CultureInfo culture)
    {
        if (values.Count == 2 && values[0] is string first && values[1] is string last)
        {
            return $"{first} {last}";
        }
        return string.Empty;
    }
}
```

```xml
<TextBlock>
  <TextBlock.Text>
    <MultiBinding Converter="{x:Static local:FullNameConverter.Instance}">
      <Binding Path="FirstName"/>
      <Binding Path="LastName"/>
    </MultiBinding>
  </TextBlock.Text>
</TextBlock>
```

**绑定验证与错误处理**

```csharp
// ViewModel中的验证
using CommunityToolkit.Mvvm.ComponentModel;
using System.ComponentModel.DataAnnotations;

public partial class UserViewModel : ObservableObject
{
    [ObservableProperty]
    private string email = string.Empty;

    partial void OnEmailChanged(string value)
    {
        ValidateProperty(value);
    }

    private void ValidateProperty(string value)
    {
        var context = new ValidationContext(this) { MemberName = nameof(Email) };
        var results = new List<ValidationResult>();

        if (!Validator.TryValidateProperty(value, context, results))
        {
            OnPropertyChanged(nameof(EmailError));
        }
    }

    public string EmailError => "邮箱格式不正确";
}
```

```xml
<!-- XAML中显示错误 -->
<StackPanel>
  <TextBox Text="{Binding Email, Mode=TwoWay}"/>
  <TextBlock Text="{Binding EmailError}"
             Foreground="Red"
             IsVisible="{Binding HasEmailError}"/>
</StackPanel>
```
