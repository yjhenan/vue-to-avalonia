## 1.2 开发环境搭建

### 1.2.1 安装.NET SDK

Avalonia基于.NET，因此首先需要安装.NET SDK。

**步骤一：下载.NET SDK**

访问官方网站：https://dotnet.microsoft.com/download

选择最新版本的.NET SDK（建议.NET 8.0或更高版本）。

**Windows系统：**
```bash
# 使用winget安装
winget install Microsoft.DotNet.SDK.8

# 或者下载安装包直接安装
# https://dotnet.microsoft.com/download/dotnet/8.0
```

**macOS系统：**
```bash
# 使用Homebrew安装
brew install dotnet-sdk

# 或下载pkg安装包
```

**Linux系统（Ubuntu示例）：**
```bash
# 添加Microsoft包源
wget https://packages.microsoft.com/config/ubuntu/22.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb

# 安装.NET SDK
sudo apt-get update
sudo apt-get install -y dotnet-sdk-8.0
```

**验证安装：**
```bash
dotnet --version
# 输出类似：8.0.100
```

### 1.2.2 IDE选择

你可以选择以下任意一个IDE进行开发：

**Visual Studio（推荐Windows用户）**
- 下载地址：https://visualstudio.microsoft.com/
- 安装时选择「.NET桌面开发」工作负载
- 优点：功能全面、调试强大、可视化设计器
- 缺点：仅支持Windows、体积较大

**Visual Studio Code（跨平台推荐）**
- 下载地址：https://code.visualstudio.com/
- 需要安装扩展：
  - C# Dev Kit（官方扩展）
  - Avalonia for VS Code（XAML智能提示）
- 优点：轻量级、跨平台、插件丰富
- 缺点：需要手动配置

**JetBrains Rider（跨平台）**
- 下载地址：https://www.jetbrains.com/rider/
- 优点：强大的重构功能、良好的Avalonia支持
- 缺点：付费软件（有30天试用期）

### 1.2.3 创建第一个Avalonia项目

**方式一：使用命令行（推荐）**

```bash
# 安装Avalonia模板
dotnet new install Avalonia.Templates

# 创建一个空白的Avalonia应用
dotnet new avalonia.app -n MyFirstApp

# 进入项目目录
cd MyFirstApp

# 运行项目
dotnet run
```

**方式二：使用Visual Studio**

1. 打开Visual Studio
2. 选择「创建新项目」
3. 搜索「Avalonia」
4. 选择「Avalonia Application」
5. 配置项目名称和位置
6. 点击「创建」

**方式三：使用VS Code**

```bash
# 创建项目
dotnet new avalonia.app -n MyFirstApp

# 用VS Code打开
code MyFirstApp
```

### 1.2.4 项目结构解析

让我们对比一下Avalonia项目和Vue CLI项目的结构：

**Avalonia项目结构：**
```
MyFirstApp/
├── App.axaml                 # 应用入口XAML
├── App.axaml.cs              # 应用入口逻辑（Code-behind）
├── MainWindow.axaml          # 主窗口XAML
├── MainWindow.axaml.cs       # 主窗口逻辑（Code-behind）
├── MyFirstApp.csproj         # 项目文件（类似package.json）
├── Program.cs                # 程序入口点
└── Properties/
    └── launchSettings.json   # 启动配置
```

**Vue CLI项目结构：**
```
my-vue-app/
├── public/
│   └── index.html           # HTML入口
├── src/
│   ├── App.vue              # 根组件
│   ├── main.js              # 应用入口
│   └── components/          # 组件目录
├── package.json             # 依赖配置
└── vite.config.js           # 构建配置
```

**关键文件对照：**

| Avalonia | Vue | 说明 |
|----------|-----|------|
| `App.axaml` | `App.vue` | 应用根组件 |
| `MainWindow.axaml` | `views/*.vue` | 页面/窗口视图 |
| `*.axaml.cs` | `<script>` 标签内 | 视图逻辑代码 |
| `MyFirstApp.csproj` | `package.json` | 项目配置和依赖 |
| `Program.cs` | `main.js` | 程序启动入口 |

### 1.2.5 第一个Avalonia程序

让我们创建一个简单的「Hello World」应用。

**MainWindow.axaml：**
```xml
<Window xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:vm="using:MyFirstApp.ViewModels"
        xmlns:d="http://schemas.microsoft.com/expression/blend/2008"
        xmlns:mc="http://schemas.openxmlformats.org/markup-compatibility/2006"
        mc:Ignorable="d" d:DesignWidth="800" d:DesignHeight="450"
        x:Class="MyFirstApp.MainWindow"
        Title="MyFirstApp"
        Width="400"
        Height="300">

    <StackPanel HorizontalAlignment="Center"
                VerticalAlignment="Center"
                Spacing="20">
        <TextBlock Text="Hello, Avalonia!"
                   FontSize="24"
                   HorizontalAlignment="Center"/>
        <TextBox Watermark="请输入你的名字"
                 Width="200"/>
        <Button Content="点击问候"
                Width="150"/>
    </StackPanel>

</Window>
```

**运行效果：**

会显示一个窗口，包含：
- 一个标题文本「Hello, Avalonia!」
- 一个输入框（占位符文字「请输入你的名字」）
- 一个按钮「点击问候」

**对应Vue实现：**
```vue
<template>
  <div class="container">
    <h1>Hello, Vue!</h1>
    <input v-model="name" placeholder="请输入你的名字" />
    <button @click="greet">点击问候</button>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const name = ref('')

function greet() {
  alert(`你好，${name.value}！`)
}
</script>

<style scoped>
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  gap: 20px;
}

h1 {
  font-size: 24px;
}

input, button {
  width: 200px;
}
</style>
```

### 1.2.6 运行和调试

**运行项目：**
```bash
# 命令行运行
dotnet run

# 或者使用热重载（Hot Reload）
dotnet watch
```

**调试项目：**

- **Visual Studio**：按F5启动调试，支持断点、变量监视等
- **VS Code**：按F5，需要配置`.vscode/launch.json`
- **Rider**：点击调试按钮或Shift+F9

**热重载（Hot Reload）：**

Avalonia支持XAML热重载，修改XAML文件后无需重启应用：

```bash
dotnet watch
```

修改`.axaml`文件后保存，界面会自动更新。这类似Vue的热更新。
