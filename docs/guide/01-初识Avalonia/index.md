# 第一章：初识Avalonia——从Web到桌面

作为Vue开发者，你已经熟悉了Web应用的开发方式。Avalonia是一个跨平台的.NET桌面UI框架，使用XAML描述界面、C#编写逻辑。本章将带你认识Avalonia、搭建开发环境，并通过技术栈对比速览表快速建立从Web到桌面的认知映射。
