# 第十一章：TodoMVC完整实现——串联所有知识点

TodoMVC是经典的TODO应用实现，涵盖了增删改查、过滤、状态管理等核心功能。本章将完整实现TodoMVC，并对比Vue实现，串联前面学习的所有知识点。

## 项目概述

### 功能需求

- ✅ 添加待办事项
- ✅ 编辑待办事项（双击编辑）
- ✅ 删除待办事项
- ✅ 标记完成/未完成
- ✅ 全部标记完成/未完成
- ✅ 过滤显示（全部/待办/已完成）
- ✅ 显示待办数量
- ✅ 清除已完成项
- ✅ 数据持久化

### 技术栈

**Vue版本：**
- Vue 3 + Composition API
- Pinia（状态管理）
- LocalStorage（持久化）

**Avalonia版本：**
- Avalonia UI
- CommunityToolkit.Mvvm（MVVM）
- JSON文件（持久化）
