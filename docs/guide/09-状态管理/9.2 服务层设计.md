## 9.2 服务层设计

### 9.2.1 单例模式

```csharp
// 单例服务（类似Pinia store的全局状态）
services.AddSingleton<UserService>();
services.AddSingleton<CartService>();
services.AddSingleton<ThemeService>();

// 每次请求创建新实例
services.AddTransient<LoginViewModel>();

// 作用域单例（同一个ViewModel内共享）
services.AddScoped<UnitOfWork>();
```

### 9.2.2 服务间通信

```csharp
// 使用事件总线或CommunityToolkit.Mvvm的WeakReferenceMessenger
using CommunityToolkit.Mvvm.Messaging;

// 定义消息类型
public record UserLoggedInMessage(string UserName);

// 发送事件
WeakReferenceMessenger.Default.Send(new UserLoggedInMessage("Alice"));

// 订阅事件
WeakReferenceMessenger.Default.Register<UserLoggedInMessage>(this, (r, m) =>
{
    Console.WriteLine($"{m.UserName} logged in");
});
```
