## 9.2 服务层设计

### 9.2.1 单例模式

```csharp
// 单例服务（类似Pinia store的全局状态）
services.AddSingleton<UserService>();
services.AddSingleton<CartService>();
services.AddSingleton<ThemeService>();

// 每次请求创建新实例
services.AddTransient<LoginViewModel>();

// 作用域单例（同一个ViewModel内共享）
services.AddScoped<UnitOfWork>();
```

### 9.2.2 服务间通信

```csharp
// 使用事件总线或ReactiveUI的消息总线
public class EventBus
{
    private readonly Subject<object> eventStream = new();

    public IObservable<T> Listen<T>()
    {
        return eventStream
            .Where(e => e is T)
            .Select(e => (T)e);
    }

    public void Publish<T>(T event)
    {
        eventStream.OnNext(event);
    }
}

// 使用
services.AddSingleton<EventBus>();

// 发布事件
eventBus.Publish(new UserLoggedInEvent { UserName = "Alice" });

// 订阅事件
eventBus.Listen<UserLoggedInEvent>()
    .Subscribe(e => Console.WriteLine($"{e.UserName} logged in"));
```
