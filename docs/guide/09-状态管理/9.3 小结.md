## 9.3 小结

本章学习了Avalonia的状态管理：

✅ **Service + DI**：替代Vuex/Pinia的全局状态管理
✅ **单例模式**：服务的生命周期管理
✅ **服务通信**：事件总线和ReactiveUI

**状态管理对比：**

| Vuex/Pinia | Avalonia | 说明 |
|-----------|----------|------|
| `defineStore` | `Service` | 状态容器 |
| `state` | 属性 | 状态数据 |
| `getters` | 计算属性 | 派生状态 |
| `actions` | 方法 | 修改状态 |
| `modules` | 多个Service | 模块化 |
| `$subscribe` | `WhenAnyValue` | 监听变化 |
| 插件 | 中间件 | 扩展功能 |

**下一章将学习：**调试与工具链，如何高效开发和调试Avalonia应用。
