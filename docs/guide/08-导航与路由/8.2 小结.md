## 8.2 小结

本章学习了Avalonia的导航模式：

✅ **窗口导航**：Show/ShowDialog打开新窗口
✅ **ContentControl导航**：通过切换视图实现页面导航，支持前进/后退
✅ **视图切换**：ContentControl + ViewModel，类似SPA

**导航方式对比：**

| Vue Router | Avalonia | 说明 |
|-----------|----------|------|
| `<router-view>` | `ContentControl` | 路由出口 |
| `<router-link>` | `Button` + Command | 导航链接 |
| `router.push()` | ViewModel.Navigate() | 编程式导航 |
| `router.back()` | ViewModel.GoBack() | 后退 |
| `router.forward()` | ViewModel.GoForward() | 前进 |
| 路由守卫 | 手动实现 | 权限控制 |
| 嵌套路由 | 嵌套ContentControl | 嵌套导航 |
