## 8.2 小结

本章学习了Avalonia 12的导航模式：

✅ **窗口导航**：Show/ShowDialog打开新窗口
✅ **NavigationPage导航**：Avalonia 12 原生栈式导航，Push/Pop/Replace
✅ **TabbedPage导航**：Tab式多页面，类似移动端底部导航栏
✅ **DrawerPage导航**：侧边栏抽屉导航，类似汉堡菜单
✅ **ContentControl切换**：简单场景的手动视图切换

**导航方式对比：**

| Vue Router | Avalonia 12 | 说明 |
|-----------|-------------|------|
| `<router-view>` | `<NavigationPage>` | 导航出口 |
| `<router-link>` | `Button` + Click/Command | 导航链接 |
| `router.push()` | `Navigation.PushAsync()` | 前进导航 |
| `router.back()` | `Navigation.PopAsync()` | 后退 |
| `router.replace()` | `Navigation.ReplaceAsync()` | 替换当前 |
| `beforeEach` 守卫 | `OnNavigating` | 导航拦截（可取消） |
| `afterEach` 守卫 | `OnNavigatedTo` | 导航后回调 |
| Tab式路由 | `<TabbedPage>` | 多Tab导航 |
| 侧边栏布局 | `<DrawerPage>` | 抽屉导航 |
