# 第八章：导航与路由——从Vue Router到Avalonia导航

Vue Router是Vue生态中不可或缺的路由库，提供了声明式路由、嵌套路由、路由守卫等功能。Avalonia 12 引入了完整的页面导航系统（NavigationPage、TabbedPage、DrawerPage），提供了类似 Vue Router 的导航体验。

## 8.1 Avalonia中的导航模式

### 8.1.1 窗口导航（Window Navigation）

**最基础的导航方式：打开新窗口**

**Vue实现（类似window.open）：**

```javascript
// 打开新页面（浏览器标签页）
window.open('/settings', '_blank')
```

**Avalonia实现：**

```csharp
// 打开新窗口
var settingsWindow = new SettingsWindow();
settingsWindow.Show();  // 非模态窗口

// 或者模态窗口
await settingsWindow.ShowDialog(this);
```

**示例：从主窗口打开设置窗口**

```csharp
// MainWindow.axaml.cs
public partial class MainWindow : Window
{
    public void OpenSettings()
    {
        var settingsWindow = new SettingsWindow();
        settingsWindow.Show();
    }

    public async Task OpenSettingsModal()
    {
        var settingsWindow = new SettingsWindow();
        await settingsWindow.ShowDialog(this);
    }
}
```

### 8.1.2 页面导航（NavigationPage + ContentPage）

> **Avalonia 12 新特性：** Avalonia 12 引入了完整的页面导航系统，包括 `ContentPage`、`NavigationPage`、`TabbedPage` 和 `DrawerPage`，提供类似 Vue Router 的声明式导航体验。

**类似Vue Router的页面导航**

**Vue Router：**

```vue
<template>
  <div>
    <nav>
      <router-link to="/home">首页</router-link>
      <router-link to="/settings">设置</router-link>
    </nav>
    <router-view />
  </div>
</template>

<script setup>
import { useRouter } from 'vue-router'
const router = useRouter()
</script>
```

**Avalonia 12 NavigationPage导航：**

```xml
<!-- MainWindow.axaml -->
<Window xmlns="https://github.com/avaloniaui">
  <NavigationPage x:Name="NavPage">
    <ContentPage Header="首页">
      <StackPanel Margin="20" Spacing="10">
        <TextBlock Text="首页" FontSize="24" FontWeight="Bold"/>
        <TextBlock Text="欢迎来到首页"/>
        <Button Content="前往设置" Click="OnGoToSettings"/>
      </StackPanel>
    </ContentPage>
  </NavigationPage>
</Window>
```

```csharp
// MainWindow.axaml.cs
using Avalonia.Controls;

public partial class MainWindow : Window
{
    public MainWindow()
    {
        InitializeComponent();
    }

    private async void OnGoToSettings(object? sender, EventArgs e)
    {
        // 类似 Vue Router 的 router.push('/settings')
        var navPage = this.Find<NavigationPage>("NavPage");
        await navPage.Navigation.PushAsync(new SettingsPage());
    }
}
```

**定义页面（ContentPage）：**

```xml
<!-- Views/SettingsPage.axaml -->
<ContentPage xmlns="https://github.com/avaloniaui"
             Header="设置">
  <StackPanel Margin="20" Spacing="10">
    <TextBlock Text="设置" FontSize="24" FontWeight="Bold"/>
    <TextBlock Text="这里是设置内容"/>
    <Button Content="返回" Click="OnGoBack"/>
  </StackPanel>
</ContentPage>
```

**导航操作（类似 Vue Router API）：**

```csharp
// 前进导航（类似 router.push）
await Navigation.PushAsync(new DetailsPage());

// 后退（类似 router.back()）
await Navigation.PopAsync();

// 回到根页面
await Navigation.PopToRootAsync();

// 替换当前页面（类似 router.replace）
await Navigation.ReplaceAsync(new HomePage());
```

**页面生命周期（类似 Vue Router 导航守卫）：**

```csharp
public class HomePage : ContentPage
{
    // 类似 Vue Router 的 beforeEach 守卫
    protected override void OnNavigating(NavigatingEventArgs args)
    {
        if (HasUnsavedChanges)
            args.Cancel = true;  // 取消导航
    }

    // 类似 Vue Router 的 afterEach 守卫
    protected override void OnNavigatedTo(NavigatedToEventArgs args)
    {
        RefreshData();  // 页面出现时刷新
    }

    // 离开页面时
    protected override void OnNavigatedFrom(NavigatedFromEventArgs args)
    {
        SaveState();  // 保存状态
    }
}
```

**导航对比表：**

| Vue Router | Avalonia 12 | 说明 |
|-----------|-------------|------|
| `router.push('/path')` | `Navigation.PushAsync(page)` | 前进导航 |
| `router.back()` | `Navigation.PopAsync()` | 后退 |
| `router.replace('/path')` | `Navigation.ReplaceAsync(page)` | 替换当前 |
| `beforeEach` 守卫 | `OnNavigating` | 导航前拦截 |
| `afterEach` 守卫 | `OnNavigatedTo` | 导航后回调 |
| `<router-view>` | `<NavigationPage>` | 导航出口 |

### 8.1.3 Tab导航（TabbedPage）

**类似Vue Router的tab导航：**

```xml
<!-- 底部Tab导航（类似移动端的 TabBar 路由） -->
<TabbedPage xmlns="https://github.com/avaloniaui"
            TabPlacement="Bottom">
  <ContentPage Header="首页" Icon="/Assets/home.png">
    <StackPanel Margin="16">
      <TextBlock Text="首页内容"/>
    </StackPanel>
  </ContentPage>

  <ContentPage Header="搜索" Icon="/Assets/search.png">
    <StackPanel Margin="16">
      <TextBlock Text="搜索内容"/>
    </StackPanel>
  </ContentPage>

  <ContentPage Header="我的" Icon="/Assets/user.png">
    <StackPanel Margin="16">
      <TextBlock Text="个人中心"/>
    </StackPanel>
  </ContentPage>
</TabbedPage>
```

**TabPlacement 选项：**
- `Top` — 顶部标签栏
- `Bottom` — 底部标签栏（移动端常用）
- `Left` / `Right` — 侧边标签栏

每个 Tab 拥有独立的导航栈，切换 Tab 时导航状态会保留。

### 8.1.4 侧边栏导航（DrawerPage）

**类似移动端的汉堡菜单导航：**

```xml
<DrawerPage xmlns="https://github.com/avaloniaui"
            Header="我的应用"
            DrawerLength="280">
  <DrawerPage.Drawer>
    <StackPanel Spacing="4" Margin="8">
      <Button Content="首页" Click="OnHomeClicked"
              HorizontalAlignment="Stretch"/>
      <Button Content="设置" Click="OnSettingsClicked"
              HorizontalAlignment="Stretch"/>
      <Button Content="关于" Click="OnAboutClicked"
              HorizontalAlignment="Stretch"/>
    </StackPanel>
  </DrawerPage.Drawer>

  <TextBlock Text="主内容区域" Margin="16"/>
</DrawerPage>
```

**DrawerBehavior 选项：**
- `Overlay` — 抽屉覆盖在内容上方
- `Split` — 抽屉推开内容（桌面端常用）
- `CompactOverlay` / `CompactInline` — 紧凑模式

### 8.1.5 手动视图切换（ContentControl）

对于不需要 NavigationPage 导航栈的简单场景，仍可使用 ContentControl 手动切换：

```csharp
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;

public partial class MainViewModel : ObservableObject
{
    [ObservableProperty]
    private UserControl? currentView;

    [RelayCommand]
    private void Navigate(string pageName)
    {
        CurrentView = pageName switch
        {
            "Home" => new HomeView(),
            "Settings" => new SettingsView(),
            _ => new HomeView()
        };
    }
}
```

```xml
<!-- 简单的内容切换 -->
<ContentControl Content="{Binding CurrentView}"/>
```
