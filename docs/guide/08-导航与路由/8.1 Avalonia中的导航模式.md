# 第八章：导航与路由——从Vue Router到Avalonia导航

Vue Router是Vue生态中不可或缺的路由库，提供了声明式路由、嵌套路由、路由守卫等功能。Avalonia作为桌面应用框架，没有内置路由系统，但我们可以通过多种方式实现类似的导航功能。

## 8.1 Avalonia中的导航模式

### 8.1.1 窗口导航（Window Navigation）

**最基础的导航方式：打开新窗口**

**Vue实现（类似window.open）：**

```javascript
// 打开新页面（浏览器标签页）
window.open('/settings', '_blank')
```

**Avalonia实现：**

```csharp
// 打开新窗口
var settingsWindow = new SettingsWindow();
settingsWindow.Show();  // 非模态窗口

// 或者模态窗口
await settingsWindow.ShowDialog(this);
```

**示例：从主窗口打开设置窗口**

```csharp
// MainWindow.axaml.cs
public partial class MainWindow : Window
{
    public void OpenSettings()
    {
        var settingsWindow = new SettingsWindow();
        settingsWindow.Show();
    }

    public async Task OpenSettingsModal()
    {
        var settingsWindow = new SettingsWindow();
        await settingsWindow.ShowDialog(this);
    }
}
```

### 8.1.2 页面导航（Frame + Page）

**类似Vue Router的页面导航**

**Vue Router：**

```vue
<template>
  <div>
    <nav>
      <router-link to="/home">首页</router-link>
      <router-link to="/settings">设置</router-link>
    </nav>

    <router-view />
  </div>
</template>

<script setup>
import { useRouter } from 'vue-router'

const router = useRouter()

function navigate(path) {
  router.push(path)
}
</script>
```

**Avalonia Frame导航：**

```xml
<!-- MainWindow.axaml -->
<Window xmlns="https://github.com/avaloniaui">
  <DockPanel>
    <!-- 导航菜单 -->
    <StackPanel DockPanel.Dock="Left" Width="200" Spacing="10" Margin="10">
      <Button Content="首页"
              Command="{Binding NavigateCommand}"
              CommandParameter="Home"/>
      <Button Content="设置"
              Command="{Binding NavigateCommand}"
              CommandParameter="Settings"/>
      <Button Content="关于"
              Command="{Binding NavigateCommand}"
              CommandParameter="About"/>
    </StackPanel>

    <!-- Frame（类似router-view） -->
    <Frame x:Name="NavigationFrame"
           Background="#f5f5f5"/>
  </DockPanel>
</Window>
```

```csharp
// MainWindow.axaml.cs
using Avalonia.Controls;
using MyApp.Views;

public partial class MainWindow : Window
{
    private readonly Frame frame;

    public MainWindow()
    {
        InitializeComponent();
        frame = this.Find<Frame>("NavigationFrame");

        // 默认导航到首页
        NavigateTo("Home");
    }

    public void NavigateTo(string pageName)
    {
        var page = pageName switch
        {
            "Home" => new HomePage(),
            "Settings" => new SettingsPage(),
            "About" => new AboutPage(),
            _ => new HomePage()
        };

        // 导航（支持前进/后退）
        frame.Navigate(page);
    }
}

// 使用ViewModel
public class MainViewModel : ReactiveObject
{
    private readonly Action<string> navigateAction;

    public ReactiveCommand<string, Unit> NavigateCommand { get; }

    public MainViewModel(Action<string> navigate)
    {
        navigateAction = navigate;
        NavigateCommand = ReactiveCommand.Create<string>(Navigate);
    }

    private void Navigate(string page) => navigateAction(page);
}
```

**定义Page：**

```xml
<!-- Pages/HomePage.axaml -->
<Page xmlns="https://github.com/avaloniaui"
```

**前进/后退导航：**

```csharp
// 后退
if (frame.CanGoBack)
{
    frame.GoBack();
}

// 前进
if (frame.CanGoForward)
{
    frame.GoForward();
}

// 导航历史
varBackStack = frame.BackStack;  // 后退栈
var forwardStack = frame.ForwardStack;  // 前进栈
```

### 8.1.3 视图切换策略（ContentControl）

**更灵活的单页应用（SPA）风格**

```csharp
// ViewModels/MainViewModel.cs
using ReactiveUI;
using System.Collections.ObjectModel;

public class MainViewModel : ReactiveObject
{
    private UserControl? currentView;

    public UserControl? CurrentView
    {
        get => currentView;
        set => this.RaiseAndSetIfChanged(ref currentView, value);
    }

    public ReactiveCommand<string, Unit> NavigateCommand { get; }

    // 导航历史
    private readonly ObservableCollection<UserControl> navigationHistory = new();
    private int currentIndex = -1;

    public MainViewModel()
    {
        NavigateCommand = ReactiveCommand.Create<string>(Navigate);
        Navigate("Home");  // 默认页
    }

    private void Navigate(string viewName)
    {
        var view = CreateView(viewName);

        // 添加到历史记录
        if (currentIndex < navigationHistory.Count - 1)
        {
            // 清除前进历史
            for (int i = navigationHistory.Count - 1; i > currentIndex; i--)
            {
                navigationHistory.RemoveAt(i);
            }
        }

        navigationHistory.Add(view);
        currentIndex++;
        CurrentView = view;
    }

    private UserControl CreateView(string viewName)
    {
        return viewName switch
        {
            "Home" => new HomeView(),
            "Settings" => new SettingsView(),
            "About" => new AboutView(),
            _ => new HomeView()
        };
    }

    public bool CanGoBack => currentIndex > 0;
    public bool CanGoForward => currentIndex < navigationHistory.Count - 1;

    public void GoBack()
    {
        if (CanGoBack)
        {
            currentIndex--;
            CurrentView = navigationHistory[currentIndex];
        }
    }

    public void GoForward()
    {
        if (CanGoForward)
        {
            currentIndex++;
            CurrentView = navigationHistory[currentIndex];
        }
    }
}
```

```xml
<!-- MainWindow.axaml -->
<Window xmlns:vm="clr-namespace:MyApp.ViewModels"
        x:DataType="vm:MainViewModel">

  <DockPanel>
    <!-- 工具栏 -->
    <ToolBar DockPanel.Dock="Top">
      <Button Content="←"
              Command="{Binding BackCommand}"
              IsEnabled="{Binding CanGoBack}"/>
      <Button Content="→"
              Command="{Binding ForwardCommand}"
              IsEnabled="{Binding CanGoForward}"/>
    </ToolBar>

    <!-- 侧边导航 -->
    <StackPanel DockPanel.Dock="Left" Width="200">
      <Button Command="{Binding NavigateCommand}"
              CommandParameter="Home">首页</Button>
      <Button Command="{Binding NavigateCommand}"
              CommandParameter="Settings">设置</Button>
    </StackPanel>

    <!-- 内容区域（类似router-view） -->
    <ContentControl Content="{Binding CurrentView}"/>
  </DockPanel>

</Window>
```
