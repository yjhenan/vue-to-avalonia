# 第八章：导航与路由——从Vue Router到Avalonia导航

Vue Router是Vue生态中不可或缺的路由库，提供了声明式路由、嵌套路由、路由守卫等功能。Avalonia作为桌面应用框架，没有内置路由系统，但我们可以通过多种方式实现类似的导航功能。

## 8.1 Avalonia中的导航模式

### 8.1.1 窗口导航（Window Navigation）

**最基础的导航方式：打开新窗口**

**Vue实现（类似window.open）：**

```javascript
// 打开新页面（浏览器标签页）
window.open('/settings', '_blank')
```

**Avalonia实现：**

```csharp
// 打开新窗口
var settingsWindow = new SettingsWindow();
settingsWindow.Show();  // 非模态窗口

// 或者模态窗口
await settingsWindow.ShowDialog(this);
```

**示例：从主窗口打开设置窗口**

```csharp
// MainWindow.axaml.cs
public partial class MainWindow : Window
{
    public void OpenSettings()
    {
        var settingsWindow = new SettingsWindow();
        settingsWindow.Show();
    }

    public async Task OpenSettingsModal()
    {
        var settingsWindow = new SettingsWindow();
        await settingsWindow.ShowDialog(this);
    }
}
```

### 8.1.2 页面导航（ContentControl + UserControl）

> **注意：** Avalonia 11 没有内置 Frame/Page 导航控件（那是 WPF/UWP 的 API）。Avalonia 12 引入了新的页面导航系统，但在 11 中推荐使用 ContentControl + 手动切换的方式。

**类似Vue Router的页面导航**

**Vue Router：**

```vue
<template>
  <div>
    <nav>
      <router-link to="/home">首页</router-link>
      <router-link to="/settings">设置</router-link>
    </nav>
    <router-view />
  </div>
</template>

<script setup>
import { useRouter } from 'vue-router'
const router = useRouter()
function navigate(path) {
  router.push(path)
}
</script>
```

**Avalonia ContentControl导航：**

```xml
<!-- MainWindow.axaml -->
<Window xmlns="https://github.com/avaloniaui"
        xmlns:vm="clr-namespace:MyApp.ViewModels"
        x:DataType="vm:MainViewModel">
  <DockPanel>
    <!-- 导航菜单 -->
    <StackPanel DockPanel.Dock="Left" Width="200" Spacing="10" Margin="10">
      <Button Content="首页"
              Command="{Binding NavigateCommand}"
              CommandParameter="Home"/>
      <Button Content="设置"
              Command="{Binding NavigateCommand}"
              CommandParameter="Settings"/>
      <Button Content="关于"
              Command="{Binding NavigateCommand}"
              CommandParameter="About"/>
    </StackPanel>
    <!-- 内容区域（类似router-view） -->
    <ContentControl Content="{Binding CurrentView}"/>
  </DockPanel>
</Window>
```

```csharp
// ViewModels/MainViewModel.cs
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;

public partial class MainViewModel : ObservableObject
{
    [ObservableProperty]
    private UserControl? currentView;

    // 导航历史（支持前进/后退）
    private readonly List<UserControl> history = new();
    private int currentIndex = -1;

    public MainViewModel()
    {
        Navigate("Home");  // 默认页
    }

    [RelayCommand]
    private void Navigate(string pageName)
    {
        var view = CreateView(pageName);
        // 清除前进历史
        if (currentIndex < history.Count - 1)
            history.RemoveRange(currentIndex + 1, history.Count - currentIndex - 1);
        history.Add(view);
        currentIndex++;
        CurrentView = view;
    }

    private static UserControl CreateView(string name) => name switch
    {
        "Home" => new HomeView(),
        "Settings" => new SettingsView(),
        "About" => new AboutView(),
        _ => new HomeView()
    };

    public bool CanGoBack => currentIndex > 0;
    public bool CanGoForward => currentIndex < history.Count - 1;

    public void GoBack()
    {
        if (CanGoBack) { currentIndex--; CurrentView = history[currentIndex]; }
    }

    public void GoForward()
    {
        if (CanGoForward) { currentIndex++; CurrentView = history[currentIndex]; }
    }
}
```

**定义视图（UserControl）：**

```xml
<!-- Views/HomeView.axaml -->
<UserControl xmlns="https://github.com/avaloniaui"

```

### 8.1.3 视图切换策略（ContentControl）

**更灵活的单页应用（SPA）风格**

```csharp
// ViewModels/MainViewModel.cs
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;
using System.Collections.ObjectModel;

public partial class MainViewModel : ObservableObject
{
    [ObservableProperty] private UserControl? currentView;

    // 导航历史
    private readonly ObservableCollection<UserControl> navigationHistory = new();
    private int currentIndex = -1;

    public MainViewModel()
    {
        Navigate("Home");  // 默认页
    }

    [RelayCommand]
    private void Navigate(string viewName)
    {
        var view = CreateView(viewName);

        // 添加到历史记录
        if (currentIndex < navigationHistory.Count - 1)
        {
            // 清除前进历史
            for (int i = navigationHistory.Count - 1; i > currentIndex; i--)
            {
                navigationHistory.RemoveAt(i);
            }
        }

        navigationHistory.Add(view);
        currentIndex++;
        CurrentView = view;
    }

    private UserControl CreateView(string viewName)
    {
        return viewName switch
        {
            "Home" => new HomeView(),
            "Settings" => new SettingsView(),
            "About" => new AboutView(),
            _ => new HomeView()
        };
    }

    public bool CanGoBack => currentIndex > 0;
    public bool CanGoForward => currentIndex < navigationHistory.Count - 1;

    public void GoBack()
    {
        if (CanGoBack)
        {
            currentIndex--;
            CurrentView = navigationHistory[currentIndex];
        }
    }

    public void GoForward()
    {
        if (CanGoForward)
        {
            currentIndex++;
            CurrentView = navigationHistory[currentIndex];
        }
    }
}
```

```xml
<!-- MainWindow.axaml -->
<Window xmlns:vm="clr-namespace:MyApp.ViewModels"
        x:DataType="vm:MainViewModel">

  <DockPanel>
    <!-- 工具栏 -->
    <ToolBar DockPanel.Dock="Top">
      <Button Content="←"
              Command="{Binding BackCommand}"
              IsEnabled="{Binding CanGoBack}"/>
      <Button Content="→"
              Command="{Binding ForwardCommand}"
              IsEnabled="{Binding CanGoForward}"/>
    </ToolBar>

    <!-- 侧边导航 -->
    <StackPanel DockPanel.Dock="Left" Width="200">
      <Button Command="{Binding NavigateCommand}"
              CommandParameter="Home">首页</Button>
      <Button Command="{Binding NavigateCommand}"
              CommandParameter="Settings">设置</Button>
    </StackPanel>

    <!-- 内容区域（类似router-view） -->
    <ContentControl Content="{Binding CurrentView}"/>
  </DockPanel>

</Window>
```
