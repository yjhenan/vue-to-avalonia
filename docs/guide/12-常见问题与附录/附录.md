### 附录A：Vue与Avalonia速查对照表

#### 语法对照

| Vue | Avalonia | 说明 |
|-----|----------|------|
| `{{ message }}` | `{Binding Message}` | 文本绑定 |
| `v-bind:href` | `{Binding Url}` | 属性绑定 |
| `v-model` | `{Binding Prop, Mode=TwoWay}` | 双向绑定 |
| `v-if` | `IsVisible` | 条件渲染 |
| `v-for` | `ItemsControl` | 列表渲染 |
| `@click` | `Command` | 事件绑定 |
| `:class` | `Classes` | 类绑定 |
| `:style` | 属性绑定 | 样式绑定 |
| `ref` | `x:Name` | 引用元素 |
| `v-show` | `IsVisible` | 显示/隐藏 |
| `v-once` | `Mode=OneTime` | 一次性绑定 |

#### API对照

| Vue API | Avalonia API | 说明 |
|---------|-------------|------|
| `ref()` | `RaiseAndSetIfChanged` | 响应式数据 |
| `reactive()` | `ReactiveObject` | 响应式对象 |
| `computed()` | C#属性 | 计算属性 |
| `watch()` | `WhenAnyValue().Subscribe()` | 监听 |
| `methods` | Command / 方法 | 方法 |
| `props` | StyledProperty | 输入属性 |
| `emits` | RoutedEvent | 输出事件 |
| `provide/inject` | DI | 依赖注入 |
| `onMounted` | 构造函数 | 初始化 |
| `onUnmounted` | `IDisposable.Dispose` | 清理 |

#### 生命周期对照

| Vue | Avalonia | 说明 |
|-----|----------|------|
| `onBeforeMount` | 无直接对应 | 挂载前 |
| `onMounted` | 构造函数 + InitializeComponent | 挂载后 |
| `onBeforeUpdate` | 无直接对应 | 更新前 |
| `onUpdated` | PropertyChanged事件 | 更新后 |
| `onBeforeUnmount` | 无直接对应 | 卸载前 |
| `onUnmounted` | Window.Closed | 卸载后 |
| `onActivated` | Window.Activated | 激活 |
| `onDeactivated` | Window.Deactivated | 失活 |

### 附录B：C#语言特性速查（针对JS开发者）

#### 变量声明

```csharp
// var - 类型推断
var name = "Alice";  // 类似let

// 显式类型
string name = "Alice";
int age = 25;

// 常量
const int MaxAge = 150;  // 类似const
readonly int InstanceValue = 10;  // 运行时常量

// 可空类型
string? name = null;  // 可为null
```

#### 字符串

```csharp
// 字符串插值（类似模板字符串）
var message = $"Hello, {name}!";

// 逐字字符串（类似Python的r""）
var path = @"C:\Users\Alice";

// 字符串方法
name.ToUpper();
name.Contains("Ali");
name.Substring(0, 3);
string.Join(", ", items);
string.Split(", ");
```

#### 集合

```csharp
// 数组
int[] numbers = { 1, 2, 3 };

// List（类似JS数组）
var list = new List<int> { 1, 2, 3 };
list.Add(4);
list.Remove(3);

// Dictionary（类似Object）
var dict = new Dictionary<string, int>
{
    ["age"] = 25
};

// ObservableCollection（响应式集合）
var oc = new ObservableCollection<int>();
```

#### LINQ（类似JS数组方法）

```csharp
var numbers = new[] { 1, 2, 3, 4, 5 };

// map
var doubled = numbers.Select(n => n * 2);

// filter
var evens = numbers.Where(n => n % 2 == 0);

// find
var first = numbers.FirstOrDefault(n => n > 3);

// reduce
var sum = numbers.Aggregate(0, (acc, n) => acc + n);

// some
var hasEven = numbers.Any(n => n % 2 == 0);

// every
var allPositive = numbers.All(n => n > 0);

// sort
var sorted = numbers.OrderBy(n => n);
```
