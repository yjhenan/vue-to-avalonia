### 2.4.1 枚举（Enum）

### 2.4.2 结构体（Struct）

```csharp
// C#结构体（值类型）
public struct Point
{
    public double X { get; set; }
    public double Y { get; set; }

    public Point(double x, double y)
    {
        X = x;
        Y = y;
    }

    public double Distance() => Math.Sqrt(X * X + Y * Y);
}

// 结构体 vs 类
// 结构体：值类型，存储在栈上，适用于小型数据
// 类：引用类型，存储在堆上
Point p1 = new Point(3, 4);
Point p2 = p1; // 值拷贝
p2.X = 10;
// p1.X 仍然是 3
```

### 2.4.3 泛型

```csharp
// C#泛型（类似TS的泛型）
public class Repository<T> where T : class, new()
{
    private List<T> items = new();

    public void Add(T item)
    {
        items.Add(item);
    }

    public T GetById(int id)
    {
        return items[id];
    }

    public IEnumerable<T> Find(Func<T, bool> predicate)
    {
        return items.Where(predicate);
    }
}

// 使用
var userRepo = new Repository<User>();
userRepo.Add(new User { Name = "Alice" });
```

**TypeScript对比：**

```typescript
// TypeScript泛型
class Repository<T extends object> {
  private items: T[] = []

  add(item: T): void {
    this.items.push(item)
  }

  find(predicate: (item: T) => boolean): T[] {
    return this.items.filter(predicate)
  }
}
```

### 2.4.4 扩展方法

```csharp
// C#扩展方法（类似TS的prototype扩展）
public static class StringExtensions
{
    public static string Capitalize(this string str)
    {
        if (string.IsNullOrEmpty(str))
            return str;

        return char.ToUpper(str[0]) + str.Substring(1);
    }

    public static bool IsNullOrEmpty(this string str)
    {
        return string.IsNullOrEmpty(str);
    }
}

// 使用
string name = "alice";
string capitalizedName = name.Capitalize(); // "Alice"
bool empty = "".IsNullOrEmpty(); // true
```

**TypeScript对比：**

```typescript
// TypeScript（需要在interface上声明）
declare global {
  interface String {
    capitalize(): string
  }
}

String.prototype.capitalize = function () {
  return this.charAt(0).toUpperCase() + this.slice(1)
}

const name = 'alice'
const capitalized = name.capitalize()
```

### 2.4.5 异常处理

```csharp
// C#
try
{
    var result = await riskyOperation();
}
catch (InvalidOperationException ex)
{
    // 处理特定异常
    Console.WriteLine($"操作无效：{ex.Message}");
}
catch (Exception ex)
{
    // 捕获所有异常
    Console.WriteLine($"发生错误：{ex.Message}");
    throw; // 重新抛出
}
finally
{
    // 清理资源
    Console.WriteLine("清理完成");
}

// 抛出异常
if (value < 0)
    throw new ArgumentException("值不能为负数", nameof(value));
```

**JavaScript对比：**

```javascript
// JavaScript
try {
  const result = await riskyOperation()
} catch (error) {
  if (error instanceof TypeError) {
    console.log('类型错误：', error.message)
  } else {
    console.log('发生错误：', error.message)
    throw error
  }
} finally {
  console.log('清理完成')
}

// 抛出异常
if (value < 0) {
  throw new Error('值不能为负数')
}
```

## 2.5 小结

本章我们学习了C#的核心语言特性，并对比了JavaScript/TypeScript：

✅ **类型系统**：C#是静态强类型，但`var`提供了类型推断的便利

✅ **面向对象**：类、接口、继承、属性——比TS更严格的OOP

✅ **LINQ**：强大的数据查询语法，比JS数组方法更强大

✅ **异步编程**：`async/await`与JS类似，但返回`Task`

✅ **模块化**：NuGet管理包，`using`引入命名空间

✅ **其他特性**：枚举、结构体、泛型、扩展方法

**学习建议：**

1. 不要害怕静态类型，IDE会提供强大的智能提示
2. 多用LINQ，它会让你爱不释手
3. 养成良好的命名习惯（C#中方法用PascalCase）
4. 熟悉async/await，Avalonia中大量使用异步
