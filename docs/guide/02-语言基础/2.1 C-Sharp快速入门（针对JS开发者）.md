## 2.1 C#快速入门（针对JS开发者）

### 2.1.1 变量声明与类型系统

**JavaScript的动态类型：**
```javascript
// JavaScript
let name = "Alice"      // string
name = 25              // 可以改为number
const age = 25         // 常量

// TypeScript
let name: string = "Alice"
const age: number = 25
```

**C#的静态类型：**
```csharp
// C# - 类型推断
var name = "Alice";     // 编译器推断为string
// name = 25;           // ❌ 编译错误！不能改变类型

// 显式类型声明
string name = "Alice";
int age = 25;
const int MaxAge = 150; // 常量

// 可空类型（C# 8.0+）
string? nullableName = null;  // 可以为null
string nonNullableName = "Bob"; // 不能为null（默认）
```

**类型对照表：**

| JavaScript/TypeScript | C# | 说明 |
|----------------------|-----|------|
| `number` | `int`, `double`, `float`, `decimal` | C#有更细的数值类型划分 |
| `string` | `string` | 字符串（C#中string是引用类型） |
| `boolean` | `bool` | 布尔值 |
| `any` | `object` | 任意类型 |
| `unknown` | - | C#没有直接对应，用模式匹配 |
| `void` | `void` | 无返回值 |
| `null` | `null` | 空值 |
| `undefined` | - | C#没有undefined |
| `Array<T>` | `T[]`, `List<T>` | 数组/列表 |
| `Object` | `class`, `struct` | 对象 |
| - | `var` | 类型推断（类似TS的类型推断） |

**数值类型详解：**

```csharp
// 整数类型
byte b = 255;           // 0-255 (1字节)
short s = 32767;        // -32768到32767 (2字节)
int i = 2147483647;     // 整数默认类型 (4字节)
long l = 9223372036854775807L; // 大整数 (8字节)

// 浮点类型
float f = 3.14f;        // 单精度 (4字节)
double d = 3.14159265359; // 双精度，默认类型 (8字节)
decimal dec = 3.14159265358979323846m; // 高精度金融计算 (16字节)

// TypeScript对比
// let num: number = 3.14; // 只有一种number类型
```

**字符串：**

```csharp
// 字符串声明
string name = "Alice";

// 字符串插值（类似TS的模板字符串）
int age = 25;
string message = $"姓名：{name}，年龄：{age}";
// TypeScript: `姓名：${name}，年龄：${age}`

// 逐字字符串（保留格式）

第三行";
```

### 2.1.2 类和接口

**TypeScript的类和接口：**

```typescript
// TypeScript
interface IPerson {
  name: string
  age: number
  greet(): string
}

class Person implements IPerson {
  constructor(
    public name: string,
    public age: number
  ) {}

  greet(): string {
    return `Hello, I'm ${this.name}`
  }
}

const person = new Person("Alice", 25)
console.log(person.greet())
```

**C#的类和接口：**

```csharp
// C#接口
public interface IPerson
{
    string Name { get; set; }
    int Age { get; set; }
    string Greet();
}

// C#类
public class Person : IPerson
{
    // 自动属性（类似TS的public属性）
    public string Name { get; set; }
    public int Age { get; set; }

    // 构造函数
    public Person(string name, int age)
    {
        Name = name;
        Age = age;
    }

    // 方法
    public string Greet()
    {
        return $"Hello, I'm {Name}";
    }
}

// 使用
var person = new Person("Alice", 25);
Console.WriteLine(person.Greet());
```

**属性（Properties）详解：**

C#的属性是TypeScript中没有的概念，它结合了字段和方法的特点：

```csharp
public class User
{
    // 自动属性（自动生成后台字段）
    public string Name { get; set; }

    // 只读属性
    public int Id { get; } = GenerateId();

    // 私有setter
    public string Email { get; private set; }

    // 带默认值的属性
    public int Age { get; set; } = 0;

    // 计算属性（类似TS的getter）
    public bool IsAdult => Age >= 18;

    // 完整属性声明（带后台字段）
    private string _password;
    public string Password
    {
        get => _password;
        set
        {
            // 可以在setter中添加验证逻辑
            if (string.IsNullOrEmpty(value))
                throw new ArgumentException("密码不能为空");
            _password = value;
        }
    }

    private static int GenerateId() => Random.Shared.Next(1, 10000);
}
```

**TypeScript对比：**

```typescript
// TypeScript用getter/setter实现类似功能
class User {
  private _password: string

  get password(): string {
    return this._password
  }

  set password(value: string) {
    if (!value) {
      throw new Error("密码不能为空")
    }
    this._password = value
  }

  // 计算属性
  get isAdult(): boolean {
    return this.age >= 18
  }
}
```

### 2.1.3 继承和多态

**TypeScript继承：**

```typescript
// TypeScript
class Animal {
  constructor(public name: string) {}

  speak(): string {
    return `${this.name} makes a sound`
  }
}

class Dog extends Animal {
  constructor(name: string, public breed: string) {
    super(name)
  }

  speak(): string {
    return `${this.name} barks`
  }
}

const dog = new Dog("Rex", "German Shepherd")
console.log(dog.speak()) // "Rex barks"
```

**C#继承：**

```csharp
// C#
public class Animal
{
    public string Name { get; set; }

    public Animal(string name)
    {
        Name = name;
    }

    // virtual表示可以被子类重写
    public virtual string Speak()
    {
        return $"{Name} makes a sound";
    }
}

public class Dog : Animal  // 使用:表示继承
{
    public string Breed { get; set; }

    public Dog(string name, string breed) : base(name) // 调用基类构造函数
    {
        Breed = breed;
    }

    // override重写基类方法
    public override string Speak()
    {
        return $"{Name} barks";
    }
}

var dog = new Dog("Rex", "German Shepherd");
Console.WriteLine(dog.Speak()); // "Rex barks"
```

**访问修饰符对照表：**

| 修饰符 | TypeScript | C# | 说明 |
|-------|-----------|-----|------|
| `public` | ✅ | ✅ | 公开访问 |
| `private` | ✅ | ✅ | 私有访问（仅类内部） |
| `protected` | ✅ | ✅ | 受保护（类内部和子类） |
| `internal` | ❌ | ✅ | 程序集内部访问（C#特有） |
| `protected internal` | ❌ | ✅ | 程序集内部或子类可访问 |

### 2.1.4 LINQ（对比JavaScript数组方法）

LINQ（Language Integrated Query）是C#中强大的数据查询语法，类似于JavaScript的数组方法链式调用。

**基础对照：**

```javascript
// JavaScript数组方法
const numbers = [1, 2, 3, 4, 5]

// map
const doubled = numbers.map(n => n * 2)

// filter
const evens = numbers.filter(n => n % 2 === 0)

// find
const found = numbers.find(n => n > 3)

// reduce
const sum = numbers.reduce((acc, n) => acc + n, 0)

// some
const hasEven = numbers.some(n => n % 2 === 0)

// every
const allPositive = numbers.every(n => n > 0)

// sort
const sorted = [...numbers].sort((a, b) => b - a)
```

```csharp
// C# LINQ
using System.Linq;

int[] numbers = { 1, 2, 3, 4, 5 };

// Select (类似map)
var doubled = numbers.Select(n => n * 2);

// Where (类似filter)
var evens = numbers.Where(n => n % 2 == 0);

// FirstOrDefault (类似find)
var found = numbers.FirstOrDefault(n => n > 3);

// Aggregate (类似reduce)
var sum = numbers.Aggregate(0, (acc, n) => acc + n);

// Any (类似some)
var hasEven = numbers.Any(n => n % 2 == 0);

// All (类似every)
var allPositive = numbers.All(n => n > 0);

// OrderByDescending (类似sort)
var sorted = numbers.OrderByDescending(n => n);
```

**复杂查询示例：**

```javascript
// JavaScript
const users = [
  { name: "Alice", age: 25, department: "IT" },
  { name: "Bob", age: 30, department: "HR" },
  { name: "Charlie", age: 28, department: "IT" }
]

const result = users
  .filter(u => u.age >= 25 && u.department === "IT")
  .map(u => ({ ...u, salary: 50000 }))
  .sort((a, b) => b.age - a.age)
  .map(u => u.name)

console.log(result) // ["Charlie", "Alice"]
```

```csharp
// C#
var users = new[]
{
    new { Name = "Alice", Age = 25, Department = "IT" },
    new { Name = "Bob", Age = 30, Department = "HR" },
    new { Name = "Charlie", Age = 28, Department = "IT" }
};

var result = users
    .Where(u => u.Age >= 25 && u.Department == "IT")
    .Select(u => new { u.Name, u.Age, u.Department, Salary = 50000 })
    .OrderByDescending(u => u.Age)
    .Select(u => u.Name)
    .ToList(); // ["Charlie", "Alice"]

// 或者使用查询语法（SQL风格）
var result2 = from u in users
              where u.Age >= 25 && u.Department == "IT"
              orderby u.Age descending
              select u.Name;
```

**常用LINQ方法完整对照表：**

| JavaScript | C# LINQ | 说明 |
|-----------|---------|------|
| `map()` | `Select()` | 映射转换 |
| `filter()` | `Where()` | 过滤 |
| `find()` | `FirstOrDefault()` | 查找第一个 |
| `findIndex()` | `FindIndex()` | 查找索引 |
| `every()` | `All()` | 全部满足 |
| `some()` | `Any()` | 任一满足 |
| `reduce()` | `Aggregate()` | 累计计算 |
| `sort()` | `OrderBy()` / `OrderByDescending()` | 排序 |
| `reverse()` | `Reverse()` | 反转 |
| `slice()` | `Skip()` + `Take()` | 分页 |
| `concat()` | `Concat()` | 连接 |
| `includes()` | `Contains()` | 包含 |
| `indexOf()` | `IndexOf()` | 查找索引 |
| `join()` | `string.Join()` | 连接为字符串 |

**分组和聚合：**

```javascript
// JavaScript分组
const users = [
  { name: "Alice", department: "IT" },
  { name: "Bob", department: "HR" },
  { name: "Charlie", department: "IT" }
]

const grouped = users.reduce((acc, user) => {
  if (!acc[user.department]) {
    acc[user.department] = []
  }
  acc[user.department].push(user)
  return acc
}, {})
```

```csharp
// C# LINQ分组
var users = new[]
{
    new { Name = "Alice", Department = "IT" },
    new { Name = "Bob", Department = "HR" },
    new { Name = "Charlie", Department = "IT" }
};

var grouped = users
    .GroupBy(u => u.Department)
    .Select(g => new
    {
        Department = g.Key,
        Users = g.ToList(),
        Count = g.Count()
    })
    .ToList();
```
