## 2.2 异步编程

### 2.2.1 async/await对比

**JavaScript的异步：**

```javascript
// JavaScript
async function fetchUser(id) {
  try {
    const response = await fetch(`/api/users/${id}`)
    const user = await response.json()
    return user
  } catch (error) {
    console.error('获取用户失败：', error)
    throw error
  }
}

// 调用
const user = await fetchUser(1)
console.log(user.name)
```

**C#的异步：**

```csharp
// C#
using System.Net.Http;
using System.Text.Json;

public async Task<User> FetchUserAsync(int id)
{
    using var client = new HttpClient();
    try
    {
        var response = await client.GetAsync($"/api/users/{id}");
        response.EnsureSuccessStatusCode();

        var json = await response.Content.ReadAsStringAsync();
        var user = JsonSerializer.Deserialize<User>(json);
        return user;
    }
    catch (Exception ex)
    {
        Console.WriteLine($"获取用户失败：{ex.Message}");
        throw;
    }
}

// 调用
var user = await FetchUserAsync(1);
Console.WriteLine(user.Name);
```

**关键区别：**

1. **返回类型**：
   - JavaScript：`async function` 总是返回Promise
   - C#：`async` 方法必须返回`Task`或`Task<T>`

2. **命名约定**：
   - C#异步方法通常以`Async`结尾（如`FetchUserAsync`）

3. **配置**：
   - C#可以配置`ConfigureAwait(false)`来避免上下文捕获

### 2.2.2 Task vs Promise

**Promise的基础操作：**

```javascript
// JavaScript Promise
const promise = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve("完成")
  }, 1000)
})

// Promise.all
const results = await Promise.all([
  fetchUser(1),
  fetchUser(2),
  fetchUser(3)
])

// Promise.race
const result = await Promise.race([
  fetchFromServer1(),
  fetchFromServer2()
])
```

**Task的基础操作：**

```csharp
// C# Task
var task = Task.Run(() =>
{
    Thread.Sleep(1000);
    return "完成";
});

// Task.WhenAll (类似Promise.all)
var results = await Task.WhenAll(
    FetchUserAsync(1),
    FetchUserAsync(2),
    FetchUserAsync(3)
);

// Task.WhenAny (类似Promise.race)
var result = await Task.WhenAny(
    FetchFromServer1Async(),
    FetchFromServer2Async()
);
```

**Task的常用方法：**

```csharp
// 延迟执行
await Task.Delay(1000); // 类似setTimeout

// 后台运行
var task = Task.Run(() => DoHeavyWork());

// 等待完成
task.Wait(); // 同步等待（不推荐）

// 获取结果
var result = await task; // 异步等待（推荐）

// 超时控制
using var cts = new CancellationTokenSource(TimeSpan.FromSeconds(5));
try
{
    var result = await FetchUserAsync(1, cts.Token);
}
catch (OperationCanceledException)
{
    Console.WriteLine("操作超时");
}
```

### 2.2.3 常见异步模式

**并行执行：**

```javascript
// JavaScript
async function loadAllData() {
  const [users, posts, comments] = await Promise.all([
    fetchUsers(),
    fetchPosts(),
    fetchComments()
  ])

  return { users, posts, comments }
}
```

```csharp
// C#
public async Task<AllData> LoadAllDataAsync()
{
    var usersTask = FetchUsersAsync();
    var postsTask = FetchPostsAsync();
    var commentsTask = FetchCommentsAsync();

    // 并行等待所有任务完成
    await Task.WhenAll(usersTask, postsTask, commentsTask);

    return new AllData
    {
        Users = await usersTask,
        Posts = await postsTask,
        Comments = await commentsTask
    };
}
```

**顺序执行：**

```csharp
// C#顺序执行
public async Task ProcessDataAsync()
{
    var data = await FetchDataAsync();
    var processed = await ProcessAsync(data);
    await SaveAsync(processed);
}
```

**异步事件处理（Avalonia中常见）：**

```csharp
// Avalonia中的异步Command
using CommunityToolkit.Mvvm.Input;

public partial class MainViewModel
{
    [RelayCommand]
    private async Task LoadDataAsync()
    {
        IsLoading = true;
        try
        {
            var data = await _dataService.GetDataAsync();
            Items = new ObservableCollection<DataItem>(data);
        }
        finally
        {
            IsLoading = false;
        }
    }
}
```

**异步锁：**

```csharp
// C#异步锁（使用SemaphoreSlim）
private readonly SemaphoreSlim _lock = new(1, 1);

public async Task SaveDataAsync(Data data)
{
    await _lock.WaitAsync();
    try
    {
        await _repository.SaveAsync(data);
    }
    finally
    {
        _lock.Release();
    }
}
```
