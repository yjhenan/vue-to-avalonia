## 10.2 性能优化

### 10.2.1 UI虚拟化（Virtualization）

**Vue虚拟滚动：**

```vue
<template>
  <RecycleScroller
    :items="items"
    :item-size="50"
    key-field="id"
  >
    <template #default="{ item }">
      <div>{{ item.name }}</div>
    </template>
  </RecycleScroller>
</template>
```

**Avalonia虚拟化：**

```xml
<!-- 使用VirtualizingStackPanel -->
<ListBox ItemsSource="{Binding Items}"
         VirtualizationMode="Simple">  <!-- 启用虚拟化 -->
  <ListBox.ItemsPanel>
    <ItemsPanelTemplate>
      <VirtualizingStackPanel />
    </ItemsPanelTemplate>
  </ListBox.ItemsPanel>
</ListBox>

<!-- 或者DataGrid（默认虚拟化） -->
<DataGrid ItemsSource="{Binding Items}"
          AutoGenerateColumns="False">
  <DataGrid.Columns>
    <DataGridTextColumn Binding="{Binding Name}"
                        Header="名称"/>
  </DataGrid.Columns>
</DataGrid>
```

**性能对比：**

| 数据量 | 无虚拟化 | 虚拟化 | 提升 |
|-------|---------|--------|------|
| 1,000 | 200ms | 50ms | 4x |
| 10,000 | 2000ms | 80ms | 25x |
| 100,000 | 20s | 120ms | 166x |

### 10.2.2 绑定优化

**避免不必要的绑定更新：**

```csharp
// ❌ 不好：频繁触发更新
[ObservableProperty]
private string searchText = string.Empty;

partial void OnSearchTextChanged(string value)
{
    // 每次输入都触发搜索
    PerformSearch(value);
}

// ✅ 好：使用防抖（通过 partial method + DispatcherTimer）
public partial class MainViewModel : ObservableObject
{
    [ObservableProperty]
    private string searchText = string.Empty;

    private DispatcherTimer? _debounceTimer;

    partial void OnSearchTextChanged(string value)
    {
        _debounceTimer?.Stop();
        _debounceTimer = new DispatcherTimer { Interval = TimeSpan.FromMilliseconds(500) };
        _debounceTimer.Tick += (s, e) =>
        {
            _debounceTimer.Stop();
            PerformSearch(value);
        };
        _debounceTimer.Start();
    }
}
```

**使用OneTime绑定：**

```xml
<!-- 只绑定一次，不监听变化（提升性能） -->
<TextBlock Text="{Binding Title, Mode=OneTime}"/>
```

### 10.2.3 异步加载

**Vue懒加载：**

```javascript
const AsyncComponent = defineAsyncComponent(() =>
  import('./HeavyComponent.vue')
)
```

**Avalonia异步加载：**

```csharp
public partial class MainViewModel : ObservableObject
{
    [ObservableProperty]
    private bool isLoading = false;

    public async Task LoadDataAsync()
    {
        IsLoading = true;
        try
        {
            await Task.Run(() =>
            {
                // 在后台线程加载大量数据
                var data = HeavyOperation();
                Avalonia.Threading.Dispatcher.UIThread.Post(() =>
                {
                    Items = new ObservableCollection<DataItem>(data);
                });
            });
        }
        finally
        {
            IsLoading = false;
        }
    }
}
```
