## 10.3 测试

### 10.3.1 单元测试

**使用xUnit：**

```bash
dotnet add package xunit
dotnet add package xunit.runner.visualstudio
```

```csharp
// Tests/MainViewModelTests.cs
using Xunit;
using MyApp.ViewModels;

public class MainViewModelTests
{
    [Fact]
    public void IncrementCommand_IncreasesCount()
    {
        // Arrange
        var viewModel = new MainViewModel();

        // Act
        viewModel.IncrementCommand.Execute().Subscribe();

        // Assert
        Assert.Equal(1, viewModel.Count);
    }

    [Theory]
    [InlineData(0, true)]   // count=0, canDecrement=false
    [InlineData(5, false)]  // count=5, canDecrement=true
    public void DecrementCommand_CanExecute_BasedOnCount(int count, bool expectedCanExecute)
    {
        // Arrange
        var viewModel = new MainViewModel { Count = count };

        // Act
        var canExecute = viewModel.DecrementCommand.CanExecute.FirstAsync().Wait();

        // Assert
        Assert.Equal(!expectedCanExecute, canExecute);
    }
}
```

**对比Vitest：**

| Vitest | xUnit | 说明 |
|--------|-------|------|
| `describe()` | `public class` | 测试套件 |
| `it()` | `[Fact]` | 测试用例 |
| `test()` | `[Fact]` | 测试用例 |
| `expect()` | `Assert` | 断言 |
| `beforeEach()` | 构造函数 | 前置钩子 |
| `afterEach()` | `IDisposable.Dispose` | 后置钩子 |
