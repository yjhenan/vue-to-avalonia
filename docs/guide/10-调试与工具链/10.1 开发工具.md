# 第十章：调试与工具链

高效的开发工具和调试技巧能极大提升开发效率。本章将介绍Avalonia开发的常用工具、调试方法和性能优化技巧。

## 10.1 开发工具

### 10.1.1 Avalonia DevTools

Avalonia DevTools是类似Vue DevTools的开发调试工具，可以实时查看和修改UI。

**安装：**

```bash
dotnet add package Avalonia.DevTools
```

**使用：**

运行应用后按`F12`打开DevTools窗口。

**功能：**

- **可视化树**：查看控件层级结构
- **属性编辑器**：实时查看和修改属性
- **数据绑定**：查看绑定路径和值
- **样式**：查看应用的样式
- **性能分析**：分析渲染性能

**对比Vue DevTools：**

| 功能 | Vue DevTools | Avalonia DevTools |
|-----|--------------|------------------|
| 组件树 | ✅ 组件层级 | ✅ 可视化树 |
| 状态查看 | ✅ Pinia/Vuex状态 | ✅ DataContext |
| 实时编辑 | ✅ 修改data | ✅ 修改属性 |
| 路由 | ✅ 路由信息 | ❌ 需手动实现 |
| 性能 | ⚠️ 基础 | ✅ 详细分析 |
| 时间旅行 | ✅ Pinia | ❌ 无 |

### 10.1.2 Visual Studio调试技巧

**断点调试：**

```csharp
// 设置断点（F9）
public void OnButtonClick()
{
    var user = userService.CurrentUser;  // 断点
    Console.WriteLine(user.Name);        // F10逐过程，F11逐语句
}
```

**条件断点：**

右键断点 → 条件 → 输入条件表达式：

```csharp
user.Name == "Alice"  // 只在条件满足时中断
```

**数据提示（DataTip）：**

鼠标悬停在变量上 → 展开查看属性 → 可以编辑值

**即时窗口（Immediate Window）：**

调试时按`Ctrl+Alt+I`打开：

```csharp
// 执行表达式
userService.CurrentUser.Name
> "Alice"

// 修改变量
userService.CurrentUser.Name = "Bob"

// 执行方法
userService.Logout()
```

**监视窗口（Watch Window）：**

调试 → 窗口 → 监视 → 监视1

```csharp
// 添加监视表达式
userService.CurrentUser
userService.IsAuthenticated
Items.Count
```

### 10.1.3 日志系统

**使用Serilog（推荐）：**

```bash
dotnet add package Serilog
dotnet add package Serilog.Sinks.Console
dotnet add package Serilog.Sinks.File
```

```csharp
// Program.cs
using Serilog;

public static void Main(string[] args)
{
    // 配置日志
    Log.Logger = new LoggerConfiguration()
        .MinimumLevel.Debug()
        .WriteTo.Console(
            outputTemplate: "[{Timestamp:HH:mm:ss} {Level:u3}] {Message:lj}{NewLine}{Exception}")
        .WriteTo.File(
            "logs/app-.log",
            rollingInterval: RollingInterval.Day,
            outputTemplate: "{Timestamp:yyyy-MM-dd HH:mm:ss.fff zzz} [{Level:u3}] {Message:lj}{NewLine}{Exception}")
        .CreateLogger();

    try
    {
        Log.Information("应用启动");
        BuildAvaloniaApp()
            .StartWithClassicDesktopLifetime(args);
    }
    catch (Exception ex)
    {
        Log.Fatal(ex, "应用崩溃");
    }
    finally
    {
        Log.CloseAndFlush();
    }
}
```

**在ViewModel中使用：**

```csharp
using Serilog;
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;

public partial class MainViewModel : ObservableObject
{
    public MainViewModel()
    {
        Log.Debug("MainViewModel初始化");
    }

    [RelayCommand]
    private async Task LoadDataAsync()
    {
        Log.Information("开始加载数据");
        try
        {
            var data = await _service.GetDataAsync();
            Log.Information("数据加载成功，共{Count}条", data.Count);
        }
        catch (Exception ex)
        {
            Log.Error(ex, "数据加载失败");
        }
    }
}
```

**对比Vue的console.log：**

| Vue | Avalonia | 说明 |
|-----|----------|------|
| `console.log()` | `Log.Information()` | 信息日志 |
| `console.warn()` | `Log.Warning()` | 警告日志 |
| `console.error()` | `Log.Error()` | 错误日志 |
| `console.debug()` | `Log.Debug()` | 调试日志 |
| 无 | `Log.Fatal()` | 致命错误 |
| 浏览器控制台 | Console/文件 | 输出位置 |
