## 6.2 主题与皮肤

### 6.2.1 Fluent主题

Avalonia提供了Fluent主题，类似Windows 11的设计风格。

**使用Fluent主题：**

```xml
<!-- App.axaml -->
<Application xmlns="https://github.com/avaloniaui"
             xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
             x:Class="MyApp.App">

  <Application.Styles>
    <!-- Fluent主题 -->
    <FluentTheme />

    <!-- 或者指定颜色模式 -->
    <FluentTheme Mode="Light"/>  <!-- Light / Dark -->
  </Application.Styles>

</Application>
```

**Fluent主题包含的样式：**

```xml
<!-- 自动包含这些控件的样式 -->
<Button Content="按钮"/>  <!-- Fluent风格按钮 -->
<TextBox Text="输入框"/>  <!-- Fluent风格输入框 -->
<CheckBox Content="复选框"/>
<RadioButton Content="单选框"/>
<ComboBox/>
<DatePicker/>
<!-- 等等所有基础控件 -->
```

**自定义Fluent主题颜色：**

```xml
<Application.Styles>
  <FluentTheme>
    <FluentTheme.Palettes>
      <ColorPaletteResources
        Accent="Blue"
        AltHigh="White"
        AltLow="White"
        AccentBase="Blue"
        <!-- 更多颜色配置 -->
      />
    </FluentTheme.Palettes>
  </FluentTheme>
</Application.Styles>
```

### 6.2.2 自定义主题

**实现亮色/暗色主题切换（完整示例）：**

**CSS变量方式（Vue）：**

```css
/* light-theme.css */
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f5f5f5;
  --text-primary: #333333;
  --text-secondary: #666666;
  --border-color: #e0e0e0;
  --accent-color: #4a90e2;
}

/* dark-theme.css */
:root {
  --bg-primary: #1a1a1a;
  --bg-secondary: #2d2d2d;
  --text-primary: #ffffff;
  --text-secondary: #cccccc;
  --border-color: #444444;
  --accent-color: #64b5f6;
}

/* 使用 */
body {
  background-color: var(--bg-primary);
  color: var(--text-primary);
}

.card {
  background-color: var(--bg-secondary);
  border: 1px solid var(--border-color);
}
```

**Avalonia实现：**

**1. 定义主题资源文件：**

```xml
<!-- Themes/LightTheme.axaml -->
<ResourceDictionary xmlns="https://github.com/avaloniaui"
                    xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">

  <SolidColorBrush x:Key="BackgroundPrimary" Color="#ffffff"/>
  <SolidColorBrush x:Key="BackgroundSecondary" Color="#f5f5f5"/>
  <SolidColorBrush x:Key="ForegroundPrimary" Color="#333333"/>
  <SolidColorBrush x:Key="ForegroundSecondary" Color="#666666"/>
  <SolidColorBrush x:Key="BorderColor" Color="#e0e0e0"/>
  <SolidColorBrush x:Key="AccentColor" Color="#4a90e2"/>

</ResourceDictionary>
```

```xml
<!-- Themes/DarkTheme.axaml -->
<ResourceDictionary xmlns="https://github.com/avaloniaui"
                    xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">

  <SolidColorBrush x:Key="BackgroundPrimary" Color="#1a1a1a"/>
  <SolidColorBrush x:Key="BackgroundSecondary" Color="#2d2d2d"/>
  <SolidColorBrush x:Key="ForegroundPrimary" Color="#ffffff"/>
  <SolidColorBrush x:Key="ForegroundSecondary" Color="#cccccc"/>
  <SolidColorBrush x:Key="BorderColor" Color="#444444"/>
  <SolidColorBrush x:Key="AccentColor" Color="#64b5f6"/>

</ResourceDictionary>
```

**2. App.axaml：**

```xml
<Application xmlns="https://github.com/avaloniaui"
             xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
             x:Class="MyApp.App">

  <Application.Resources>
    <!-- 默认使用亮色主题 -->
    <ResourceDictionary>
      <ResourceDictionary.MergedDictionaries>
        <ResourceInclude Source="/Themes/LightTheme.axaml"/>
      </ResourceDictionary.MergedDictionaries>
    </ResourceDictionary>
  </Application.Resources>

</Application>
```

**3. MainWindow.axaml：**

```xml
<Window xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:vm="clr-namespace:MyApp.ViewModels"
        x:Class="MyApp.Views.MainWindow"
        x:DataType="vm:MainViewModel"
        Title="主题切换示例"
        Width="800" Height="600"
        Background="{DynamicResource BackgroundPrimary}">

  <Window.Styles>
    <Style Selector="TextBlock">
      <Setter Property="Foreground" Value="{DynamicResource ForegroundPrimary}"/>
    </Style>
  </Window.Styles>

  <Grid RowDefinitions="Auto,*">
    <!-- 头部工具栏 -->
    <Border Grid.Row="0"
            Background="{DynamicResource BackgroundSecondary}"
            BorderBrush="{DynamicResource BorderColor}"
            BorderThickness="0,0,0,1"
            Padding="15">
      <DockPanel>
        <TextBlock DockPanel.Dock="Left"
                   Text="主题切换示例"
                   FontSize="20"
                   FontWeight="Bold"/>

        <Button DockPanel.Dock="Right"
                Content="{Binding ThemeButtonText}"
                Command="{Binding ToggleThemeCommand}"
                HorizontalAlignment="Right"/>
      </DockPanel>
    </Border>

    <!-- 主内容 -->
    <ScrollViewer Grid.Row="1">
      <StackPanel Margin="20" Spacing="15">
        <TextBlock Text="主标题"
                   FontSize="24"
                   FontWeight="Bold"/>

        <TextBlock Text="这是一段描述文本，演示主题切换效果。"
                   Foreground="{DynamicResource ForegroundSecondary}"/>

        <Border Background="{DynamicResource BackgroundSecondary}"
                BorderBrush="{DynamicResource BorderColor}"
                BorderThickness="1"
                Padding="15"
                CornerRadius="4">
          <StackPanel Spacing="10">
            <TextBlock Text="卡片标题" FontWeight="Bold"/>
            <TextBlock Text="卡片内容"
                       Foreground="{DynamicResource ForegroundSecondary}"/>
            <Button Content="操作按钮"
                    Background="{DynamicResource AccentColor}"
                    Foreground="White"
                    HorizontalAlignment="Left"/>
          </StackPanel>
        </Border>

        <TextBox Text="输入框示例"
                 Margin="0,10,0,0"/>
      </StackPanel>
    </ScrollViewer>
  </Grid>

</Window>
```

**4. 主题切换逻辑：**

```csharp
// ViewModels/MainViewModel.cs
using Avalonia;
using Avalonia.Markup.Xaml.Styling;
using Avalonia.Styling;
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;

public partial class MainViewModel : ObservableObject
{
    [ObservableProperty] private bool isDarkTheme = false;

    public string ThemeButtonText => IsDarkTheme ? "☀️ 亮色" : "🌙 暗色";

    partial void OnIsDarkThemeChanged(bool value)
    {
        OnPropertyChanged(nameof(ThemeButtonText));
        ApplyTheme(value);
    }

    [RelayCommand]
    private void ToggleTheme()
    {
        IsDarkTheme = !IsDarkTheme;
    }

    private void ApplyTheme(bool dark)
    {
        // 获取当前应用
        var app = Application.Current;
        if (app == null) return;

        // 清除现有资源
        app.Resources.MergedDictionaries.Clear();

        // 加载主题资源
        var themeUrl = dark
            ? "avares://MyApp/Themes/DarkTheme.axaml"
            : "avares://MyApp/Themes/LightTheme.axaml";

        var themeResources = AvaloniaXamlLoader.Load(
            new Uri(themeUrl)
        ) as ResourceDictionary;

        if (themeResources != null)
        {
            app.Resources.MergedDictionaries.Add(themeResources);
        }
    }
}
```

**或者使用FluentTheme内置的暗色模式：**

```csharp
private void ApplyTheme(bool dark)
{
    var app = Application.Current;
    if (app == null) return;

    // 方式1：切换FluentTheme模式
    var fluentTheme = app.Styles.OfType<FluentTheme>().FirstOrDefault();
    if (fluentTheme != null)
    {
        fluentTheme.Mode = dark ? FluentThemeMode.Dark : FluentThemeMode.Light;
    }

    // 方式2：完全替换主题
    app.Styles.Clear();
    app.Styles.Add(new FluentTheme
    {
        Mode = dark ? FluentThemeMode.Dark : FluentThemeMode.Light
    });
}
```

### 6.2.3 Material Design主题

使用Material.Avalonia库实现Material Design风格：

**1. 安装NuGet包：**

```bash
dotnet add package Material.Avalonia
```

**2. 配置Material主题：**

```xml
<!-- App.axaml -->
<Application xmlns="https://github.com/avaloniaui"
             xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
             xmlns:material="clr-namespace:Material.Styles.Themes;assembly=Material.Avalonia"
             x:Class="MyApp.App">

  <Application.Styles>
    <material:MaterialTheme BaseTheme="Light" PrimaryColor="Blue" SecondaryColor="Amber"/>
  </Application.Styles>

</Application>
```

**3. 使用Material控件：**

```xml
<Window xmlns:material="clr-namespace:Material.Controls;assembly=Material.Avalonia">
  <StackPanel Margin="20" Spacing="15">
    <!-- Material按钮 -->
    <material:Button Content="Material Button"
                     Style="{StaticResource MaterialRaisedButton}"/>

    <!-- Material卡片 -->
    <material:Card Padding="15">
      <StackPanel>
        <TextBlock Text="Card Title"
                   FontSize="18"
                   FontWeight="Bold"/>
        <TextBlock Text="Card content goes here"
                   Margin="0,10,0,0"/>
      </StackPanel>
    </material:Card>

    <!-- Material输入框 -->
    <material:TextBox Text="Material TextBox"
                      HintText="Enter text..."/>
  </StackPanel>
</Window>
```

### 6.2.4 动画与过渡

**CSS过渡动画：**

```css
.button {
  background-color: #4a90e2;
  transition: background-color 0.3s ease, transform 0.3s ease;
}

.button:hover {
  background-color: #357abd;
  transform: scale(1.05);
}

.fade-enter-active {
  animation: fadeIn 0.5s;
}

@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(-10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

**Avalonia动画：**

```xml
<Window xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">

  <Window.Styles>
    <!-- 过渡动画（类似CSS transition） -->
    <Style Selector="Button.animated">
      <Setter Property="Transitions">
        <Transitions>
          <!-- 颜色过渡 -->
          <BrushTransition Property="Background" Duration="0:0:0.3"/>

          <!-- 尺寸过渡 -->
          <DoubleTransition Property="Width" Duration="0:0:0.3" Easing="CubicEaseOut"/>

          <!-- 透明度过渡 -->
          <DoubleTransition Property="Opacity" Duration="0:0:0.2"/>

          <!-- 变换过渡 -->
          <TransformOperationsTransition Property="RenderTransform" Duration="0:0:0.3"/>
        </Transitions>
      </Setter>
    </Style>

    <!-- Hover效果 -->
    <Style Selector="Button.animated:pointerover">
      <Setter Property="Background" Value="#357abd"/>
      <Setter Property="RenderTransform" Value="scale(1.05)"/>
    </Style>
  </Window.Styles>

  <StackPanel Margin="20">
    <Button Content="悬停我"
            Classes="animated"
            Width="150"
            Height="40"
            Background="#4a90e2"
            Foreground="White"/>
  </StackPanel>

</Window>
```

**关键帧动画（类似@keyframes）：**

```xml
<Window xmlns="https://github.com/avaloniaui">
  <Window.Resources>
    <!-- 定义动画 -->
    <Animation x:Key="FadeInAnimation" Duration="0:0:0.5">
      <KeyFrame Cue="0%">
        <Setter Property="Opacity" Value="0"/>
        <Setter Property="TranslateTransform.Y" Value="-10"/>
      </KeyFrame>
      <KeyFrame Cue="100%">
        <Setter Property="Opacity" Value="1"/>
        <Setter Property="TranslateTransform.Y" Value="0"/>
      </KeyFrame>
    </Animation>
  </Window.Resources>

  <!-- 应用动画 -->
  <TextBlock Text="Hello World"
             RenderTransform="translate(0, 0)">
    <TextBlock.Styles>
      <Style Selector="TextBlock">
        <Style.Animations>
          <Animation Duration="0:0:0.5" FillMode="Forward">
            <KeyFrame Cue="0%">
              <Setter Property="Opacity" Value="0"/>
            </KeyFrame>
            <KeyFrame Cue="100%">
              <Setter Property="Opacity" Value="1"/>
            </KeyFrame>
          </Animation>
        </Style.Animations>
      </Style>
    </TextBlock.Styles>
  </TextBlock>

</Window>
```

**代码触发动画：**

```csharp
using Avalonia.Animation;
using Avalonia.Styling;

// 创建动画
var animation = new Animation
{
    Duration = TimeSpan.FromSeconds(0.5),
    Children =
    {
        new KeyFrame
        {
            Cue = new Cue(0),
            Setters = { new Setter(OpacityProperty, 0.0) }
        },
        new KeyFrame
        {
            Cue = new Cue(1),
            Setters = { new Setter(OpacityProperty, 1.0) }
        }
    }
};

// 应用到控件
await animation.RunAsync(myControl);
```
