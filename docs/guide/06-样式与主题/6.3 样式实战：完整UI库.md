## 6.3 样式实战：完整UI库

### 6.3.1 创建可复用样式库

**项目结构：**

```
MyApp.Themes/
├── Themes/
│   ├── LightTheme.axaml
│   └── DarkTheme.axaml
├── Styles/
│   ├── Buttons.axaml
│   ├── Inputs.axaml
│   ├── Cards.axaml
│   └── Texts.axaml
└── Themes.csproj
```

**Buttons.axaml：**

```xml
<ResourceDictionary xmlns="https://github.com/avaloniaui"
                    xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">

  <!-- 按钮基础样式 -->
  <Style Selector="Button.btn">
    <Setter Property="Padding" Value="12,24"/>
    <Setter Property="CornerRadius" Value="4"/>
    <Setter Property="FontWeight" Value="Medium"/>
    <Setter Property="Cursor" Value="Hand"/>
    <Setter Property="Transitions">
      <Transitions>
        <DoubleTransition Property="Opacity" Duration="0:0:0.15"/>
        <BrushTransition Property="Background" Duration="0:0:0.15"/>
      </Transitions>
    </Setter>
  </Style>

  <!-- 尺寸 -->
  <Style Selector="Button.btn-sm">
    <Setter Property="Padding" Value="6,12"/>
    <Setter Property="FontSize" Value="12"/>
  </Style>

  <Style Selector="Button.btn-lg">
    <Setter Property="Padding" Value="18,36"/>
    <Setter Property="FontSize" Value="16"/>
  </Style>

  <!-- 颜色变体 -->
  <Style Selector="Button.btn-primary">
    <Setter Property="Background" Value="#4a90e2"/>
    <Setter Property="Foreground" Value="White"/>
  </Style>

  <Style Selector="Button.btn-primary:pointerover /template/ ContentPresenter">
    <Setter Property="Background" Value="#357abd"/>
  </Style>

  <!-- 更多样式... -->

</ResourceDictionary>
```

**App.axaml引用：**

```xml
<Application.Styles>
  <!-- 引入样式库 -->
  <StyleInclude Source="/Styles/Buttons.axaml"/>
  <StyleInclude Source="/Styles/Inputs.axaml"/>
  <StyleInclude Source="/Styles/Cards.axaml"/>
  <StyleInclude Source="/Styles/Texts.axaml"/>
</Application.Styles>
```

## 6.4 小结

本章学习了Avalonia的样式和主题系统：

✅ **Style语法**：Selector和Setter的使用
✅ **选择器**：类型、类、Name、伪类等选择器
✅ **资源字典**：定义和引用样式资源
✅ **主题切换**：实现亮色/暗色主题
✅ **动画过渡**：Transitions和KeyFrame动画
✅ **UI库创建**：组织可复用样式

**与CSS对比总结：**

| CSS特性 | Avalonia对应 | 说明 |
|--------|------------|------|
| `color: red` | `Foreground="Red"` | 前景色 |
| `background: blue` | `Background="Blue"` | 背景色 |
| `font-size: 14px` | `FontSize="14"` | 字体大小 |
| `margin: 10px` | `Margin="10"` | 外边距 |
| `padding: 10px` | `Padding="10"` | 内边距 |
| `border-radius: 4px` | `CornerRadius="4"` | 圆角 |
| `:hover` | `:pointerover` | 悬停伪类 |
| `transition` | `Transitions` | 过渡动画 |
| `@keyframes` | `KeyFrame` | 关键帧动画 |
| CSS变量 | `ResourceDictionary` | 样式变量 |
