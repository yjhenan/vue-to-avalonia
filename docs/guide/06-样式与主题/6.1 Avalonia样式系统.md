# 第六章：样式与主题——从CSS到Avalonia样式

作为Vue开发者，你已经熟悉CSS的强大和灵活。Avalonia的样式系统借鉴了CSS的选择器和级联概念，但又有其独特之处。本章将帮助你理解Avalonia的样式系统，并建立从CSS到Avalonia样式的完整映射。

## 6.1 Avalonia样式系统

### 6.1.1 Style语法对比CSS

**CSS样式：**

```css
/* 基础选择器 */
button {
  background-color: #4a90e2;
  color: white;
  padding: 10px 20px;
  border-radius: 4px;
}

/* 类选择器 */
.btn-primary {
  background-color: #4a90e2;
}

/* ID选择器 */
#submit-btn {
  width: 100%;
}

/* 伪类 */
button:hover {
  background-color: #357abd;
}

button:disabled {
  opacity: 0.5;
}

/* 后代选择器 */
.container button {
  margin: 5px;
}

/* 多个选择器 */
h1, h2, h3 {
  font-weight: bold;
}
```

**Avalonia样式：**

```xml
<Window.Styles>
  <!-- 类型选择器（类似标签选择器） -->
  <Style Selector="Button">
    <Setter Property="Background" Value="#4a90e2"/>
    <Setter Property="Foreground" Value="White"/>
    <Setter Property="Padding" Value="10,20"/>
    <Setter Property="CornerRadius" Value="4"/>
  </Style>

  <!-- 类选择器 -->
  <Style Selector="Button.btn-primary">
    <Setter Property="Background" Value="#4a90e2"/>
  </Style>

  <!-- Name选择器（类似ID选择器） -->
  <Style Selector="#submit-btn">
    <Setter Property="Width" Value="100"/>
  </Style>

  <!-- 伪类选择器 -->
  <Style Selector="Button:pointerover">
    <Setter Property="Background" Value="#357abd"/>
  </Style>

  <Style Selector="Button:disabled">
    <Setter Property="Opacity" Value="0.5"/>
  </Style>

  <!-- 后代选择器 -->
  <Style Selector="StackPanel.container Button">
    <Setter Property="Margin" Value="5"/>
  </Style>

  <!-- 多个选择器 -->
  <Style Selector="TextBlock.h1, TextBlock.h2, TextBlock.h3">
    <Setter Property="FontWeight" Value="Bold"/>
  </Style>
</Window.Styles>
```

**语法对照表：**

| CSS选择器 | Avalonia选择器 | 示例 |
|----------|---------------|------|
| `button` | `Button` | 类型选择器（控件类型） |
| `.btn-primary` | `Button.btn-primary` | 类选择器 |
| `#submit-btn` | `#submit-btn` | Name选择器 |
| `button:hover` | `Button:pointerover` | 伪类选择器 |
| `.container button` | `StackPanel.container Button` | 后代选择器 |
| `button.primary` | `Button.primary` | 组合选择器 |
| `button, input` | `Button, TextBox` | 多选择器 |

### 6.1.2 Selector选择器详解

**常用选择器完整列表：**

```xml
<Window.Styles>
  <!-- 1. 类型选择器 -->
  <Style Selector="Button">
    <Setter Property="Background" Value="Blue"/>
  </Style>

  <!-- 2. 类选择器 -->
  <Style Selector=".primary">
    <Setter Property="Background" Value="Green"/>
  </Style>

  <!-- 3. Name选择器 -->
  <Style Selector="#myButton">
    <Setter Property="Width" Value="200"/>
  </Style>

  <!-- 4. 类型+类选择器 -->
  <Style Selector="Button.primary">
    <Setter Property="FontSize" Value="16"/>
  </Style>

  <!-- 5. 子选择器（直接子元素） -->
  <Style Selector="StackPanel > Button">
    <Setter Property="Margin" Value="10"/>
  </Style>

  <!-- 6. 后代选择器（所有后代） -->
  <Style Selector="Grid Button">
    <Setter Property="Padding" Value="5"/>
  </Style>

  <!-- 7. 伪类选择器 -->
  <!-- :pointerover（鼠标悬停） -->
  <Style Selector="Button:pointerover">
    <Setter Property="Background" Value="DarkBlue"/>
  </Style>

  <!-- :pressed（按下） -->
  <Style Selector="Button:pressed">
    <Setter Property="Background" Value="Navy"/>
  </Style>

  <!-- :disabled（禁用） -->
  <Style Selector="Button:disabled">
    <Setter Property="Opacity" Value="0.5"/>
  </Style>

  <!-- :focus（获得焦点） -->
  <Style Selector="TextBox:focus">
    <Setter Property="BorderBrush" Value="Blue"/>
    <Setter Property="BorderThickness" Value="2"/>
  </Style>

  <!-- 8. Template子元素选择器 -->
  <Style Selector="Button /template/ ContentPresenter">
    <Setter Property="Foreground" Value="White"/>
  </Style>

  <!-- 9. 逻辑运算 -->
  <!-- :not() -->
  <Style Selector="Button:not(.primary)">
    <Setter Property="Background" Value="Gray"/>
  </Style>

  <!-- 多个类 -->
  <Style Selector="Button.primary.large">
    <Setter Property="FontSize" Value="20"/>
  </Style>

  <!-- 10. 属性选择器 -->
  <Style Selector="Button[IsDefault=True]">
    <Setter Property="Background" Value="Green"/>
  </Style>

</Window.Styles>
```

**伪类完整列表：**

| CSS伪类 | Avalonia伪类 | 说明 |
|---------|------------|------|
| `:hover` | `:pointerover` | 鼠标悬停 |
| `:active` | `:pressed` | 按下 |
| `:focus` | `:focus` | 获得焦点 |
| `:disabled` | `:disabled` | 禁用 |
| `:checked` | `:checked` | 选中（CheckBox/RadioButton） |
| `:empty` | `:empty` | 无内容 |
| `:first-child` | `:first-child` | 第一个子元素 |
| `:last-child` | `:last-child` | 最后一个子元素 |
| `:nth-child(n)` | `:nth-child(n)` | 第n个子元素 |
| 无直接对应 | `:selected` | 列表选中项 |

### 6.1.3 资源字典（ResourceDictionary）

**CSS变量 vs Avalonia资源：**

**CSS变量：**

```css
:root {
  --primary-color: #4a90e2;
  --secondary-color: #6c757d;
  --font-size-base: 14px;
  --border-radius: 4px;
}

.button {
  background-color: var(--primary-color);
  font-size: var(--font-size-base);
  border-radius: var(--border-radius);
}
```

**Avalonia资源：**

```xml
<Window.Resources>
  <!-- 定义资源 -->
  <SolidColorBrush x:Key="PrimaryColor">#4a90e2</SolidColorBrush>
  <SolidColorBrush x:Key="SecondaryColor">#6c757d</SolidColorBrush>
  <x:Double x:Key="FontSizeBase">14</x:Double>
  <x:Double x:Key="BorderRadius">4</x:Double>
</Window.Resources>

<!-- 使用资源 -->
<Button Background="{DynamicResource PrimaryColor}"
        FontSize="{DynamicResource FontSizeBase}"
        CornerRadius="{DynamicResource BorderRadius}"
        Content="按钮"/>
```

**资源类型：**

```xml
<Window.Resources>
  <!-- 颜色和画刷 -->
  <Color x:Key="PrimaryColorValue">#4a90e2</Color>
  <SolidColorBrush x:Key="PrimaryBrush" Color="{StaticResource PrimaryColorValue}"/>

  <!-- 渐变画刷 -->
  <LinearGradientBrush x:Key="GradientBrush" StartPoint="0%,0%" EndPoint="100%,100%">
    <GradientStop Color="#4a90e2" Offset="0"/>
    <GradientStop Color="#9b59b6" Offset="1"/>
  </LinearGradientBrush>

  <!-- 数值 -->
  <x:Double x:Key="FontSizeSmall">12</x:Double>
  <x:Double x:Key="FontSizeNormal">14</x:Double>
  <x:Double x:Key="FontSizeLarge">18</x:Double>

  <!-- 厚度（Margin/Padding） -->
  <Thickness x:Key="MarginSmall">5</Thickness>
  <Thickness x:Key="MarginNormal">10</Thickness>
  <Thickness x:Key="PaddingButton">10,20</Thickness>

  <!-- 圆角 -->
  <CornerRadius x:Key="CornerRadiusSmall">4</CornerRadius>
  <CornerRadius x:Key="CornerRadiusLarge">8</CornerRadius>

  <!-- 字体 -->
  <FontFamily x:Key="FontFamilyNormal">Microsoft YaHei, Segoe UI</FontFamily>

  <!-- 字体粗细 -->
  <FontWeight x:Key="FontWeightBold">Bold</FontWeight>

  <!-- 字体样式 -->
  <FontStyle x:Key="FontStyleItalic">Italic</FontStyle>

  <!-- 字符串 -->
  <x:String x:Key="AppName">我的应用</x:String>

  <!-- 布尔值 -->
  <x:Boolean x:Key="IsDarkTheme">False</x:Boolean>

  <!-- 对象 -->
  <local:AppSettings x:Key="Settings"
                     Theme="Light"
                     Language="zh-CN"/>
</Window.Resources>
```

**StaticResource vs DynamicResource：**

```xml
<Window.Resources>
  <SolidColorBrush x:Key="MyBrush" Color="Red"/>
</Window.Resources>

<!-- StaticResource：编译时解析，性能更好 -->
<Border Background="{StaticResource MyBrush}"/>

<!-- DynamicResource：运行时解析，支持动态更新 -->
<Border Background="{DynamicResource MyBrush}"/>
```

```csharp
// DynamicResource可以在运行时更改
this.Resources["MyBrush"] = Brushes.Blue;  // UI会自动更新
```

### 6.1.4 样式继承与覆盖

**CSS的级联：**

```css
/* 全局样式 */
button {
  padding: 10px;
  background-color: gray;
}

/* 覆盖全局样式 */
.container button {
  background-color: blue;
}

/* 更具体的选择器优先级更高 */
.container .primary {
  background-color: green;
}
```

**Avalonia的样式级联：**

```xml
<!-- App.axaml（应用级样式） -->
<Application.Styles>
  <Style Selector="Button">
    <Setter Property="Padding" Value="10"/>
    <Setter Property="Background" Value="Gray"/>
  </Style>
</Application.Styles>

<!-- MainWindow.axaml（窗口级样式，覆盖应用级） -->
<Window.Styles>
  <Style Selector="Button">
    <Setter Property="Background" Value="Blue"/>
  </Style>
</Window.Styles>

<!-- 局部容器样式（优先级更高） -->
<StackPanel.Styles>
  <Style Selector="Button.primary">
    <Setter Property="Background" Value="Green"/>
  </Style>
</StackPanel.Styles>
```

**样式优先级规则：**

1. **就近原则**：局部样式 > 全局样式
2. **具体性**：更具体的选择器优先
3. **定义顺序**：后定义的样式覆盖先定义的
4. **内联样式**：直接设置属性优先级最高

```xml
<Window.Resources>
  <SolidColorBrush x:Key="MyBrush" Color="Blue"/>
</Window.Styles>

<Window.Styles>
  <Style Selector="Button">
    <Setter Property="Background" Value="Red"/>  <!-- 优先级1 -->
  </Style>

  <Style Selector="Button.special">
    <Setter Property="Background" Value="Green"/>  <!-- 优先级2 -->
  </Style>
</Window.Styles>

<StackPanel>
  <!-- 内联样式优先级最高 -->
  <Button Background="Yellow" Content="黄色"/>  <!-- 优先级4 -->

  <!-- 应用.special样式 -->
  <Button Classes="special" Content="绿色"/>  <!-- 优先级3 -->

  <!-- 应用基础样式 -->
  <Button Content="红色"/>  <!-- 优先级2 -->

  <!-- 使用资源 -->
  <Button Background="{StaticResource MyBrush}" Content="蓝色"/>  <!-- 优先级3 -->
</StackPanel>
```

### 6.1.5 完整示例：按钮样式库

**CSS实现（Tailwind风格）：**

```html
<style>
.btn {
  padding: 0.5rem 1rem;
  border-radius: 0.25rem;
  font-weight: 500;
  transition: all 0.15s;
  cursor: pointer;
}

.btn-sm {
  padding: 0.25rem 0.5rem;
  font-size: 0.875rem;
}

.btn-lg {
  padding: 0.75rem 1.5rem;
  font-size: 1.125rem;
}

.btn-primary {
  background-color: #4a90e2;
  color: white;
}

.btn-primary:hover {
  background-color: #357abd;
}

.btn-secondary {
  background-color: #6c757d;
  color: white;
}

.btn-secondary:hover {
  background-color: #5a6268;
}

.btn-danger {
  background-color: #dc3545;
  color: white;
}

.btn-danger:hover {
  background-color: #c82333;
}

.btn-outline {
  background-color: transparent;
  border: 1px solid currentColor;
}

.btn-rounded {
  border-radius: 9999px;
}
</style>

<!-- 使用 -->
<button class="btn btn-primary btn-lg">主要按钮</button>
<button class="btn btn-secondary btn-sm">次要按钮</button>
<button class="btn btn-danger btn-outline">危险按钮</button>
<button class="btn btn-primary btn-rounded">圆角按钮</button>
```

**Avalonia实现：**

```xml
<!-- App.axaml -->
<Application.Styles>
  <!-- 基础按钮样式 -->
  <Style Selector="Button.btn">
    <Setter Property="Padding" Value="10,20"/>
    <Setter Property="CornerRadius" Value="4"/>
    <Setter Property="FontWeight" Value="Medium"/>
    <Setter Property="Cursor" Value="Hand"/>
    <Setter Property="BorderThickness" Value="0"/>
    <Setter Property="Transitions">
      <Transitions>
        <DoubleTransition Property="Opacity" Duration="0:0:0.15"/>
      </Transitions>
    </Setter>
  </Style>

  <!-- 尺寸变体 -->
  <Style Selector="Button.btn-sm">
    <Setter Property="Padding" Value="5,10"/>
    <Setter Property="FontSize" Value="12"/>
  </Style>

  <Style Selector="Button.btn-lg">
    <Setter Property="Padding" Value="15,30"/>
    <Setter Property="FontSize" Value="16"/>
  </Style>

  <!-- 主要按钮 -->
  <Style Selector="Button.btn-primary">
    <Setter Property="Background" Value="#4a90e2"/>
    <Setter Property="Foreground" Value="White"/>
  </Style>

  <Style Selector="Button.btn-primary:pointerover /template/ ContentPresenter">
    <Setter Property="Background" Value="#357abd"/>
  </Style>

  <!-- 次要按钮 -->
  <Style Selector="Button.btn-secondary">
    <Setter Property="Background" Value="#6c757d"/>
    <Setter Property="Foreground" Value="White"/>
  </Style>

  <Style Selector="Button.btn-secondary:pointerover /template/ ContentPresenter">
    <Setter Property="Background" Value="#5a6268"/>
  </Style>

  <!-- 危险按钮 -->
  <Style Selector="Button.btn-danger">
    <Setter Property="Background" Value="#dc3545"/>
    <Setter Property="Foreground" Value="White"/>
  </Style>

  <Style Selector="Button.btn-danger:pointerover /template/ ContentPresenter">
    <Setter Property="Background" Value="#c82333"/>
  </Style>

  <!-- 轮廓按钮 -->
  <Style Selector="Button.btn-outline">
    <Setter Property="Background" Value="Transparent"/>
    <Setter Property="BorderThickness" Value="1"/>
    <Setter Property="BorderBrush" Value="{Binding $parent.Foreground}"/>
  </Style>

  <!-- 圆角按钮 -->
  <Style Selector="Button.btn-rounded">
    <Setter Property="CornerRadius" Value="9999"/>
  </Style>

</Application.Styles>
```

```xml
<!-- 使用 -->
<StackPanel Spacing="10" Margin="20">
  <Button Classes="btn btn-primary btn-lg"
          Content="主要按钮"/>

  <Button Classes="btn btn-secondary btn-sm"
          Content="次要按钮"/>

  <Button Classes="btn btn-danger btn-outline"
          Content="危险按钮"/>

  <Button Classes="btn btn-primary btn-rounded"
          Content="圆角按钮"/>
</StackPanel>
```

**动态切换Classes：**

```csharp
// ViewModel
using CommunityToolkit.Mvvm.ComponentModel;

public partial class ButtonViewModel : ObservableObject
{
    [ObservableProperty] private bool isPrimary = true;

    public string ButtonClasses => IsPrimary
        ? "btn btn-primary"
        : "btn btn-secondary";

    partial void OnIsPrimaryChanged(bool value)
    {
        OnPropertyChanged(nameof(ButtonClasses));
    }
}
```

```xml
<Button Classes="{Binding ButtonClasses}"
        Content="动态样式"/>
```

## 6.2 小结

本节学习了Avalonia样式系统的核心概念：

✅ **Style语法**：理解Selector和Setter的使用
✅ **选择器**：掌握各种选择器的语法和优先级
✅ **资源字典**：使用ResourceDictionary管理样式资源
✅ **样式继承**：理解样式级联和覆盖规则
✅ **完整示例**：实现按钮样式库

**CSS vs Avalonia样式对比：**

| 方面 | CSS | Avalonia |
|-----|-----|----------|
| **选择器** | 丰富的CSS选择器 | 类似但略有不同 |
| **级联** | 强大的级联规则 | 类似的级联机制 |
| **变量** | CSS变量 / SCSS变量 | ResourceDictionary |
| **预处理器** | Sass/Less | 无需预处理器 |
| **作用域** | 全局 / Scoped | 自动级联 |
| **动态更新** | CSS变量支持 | DynamicResource支持 |
