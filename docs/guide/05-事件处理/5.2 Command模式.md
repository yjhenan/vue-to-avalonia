## 5.2 Command模式

Command模式是Avalonia MVVM架构中处理事件的核心方式。相比于Vue的methods，Command提供了更好的可测试性、可复用性和可组合性。

### 5.2.1 ICommand接口详解

**ICommand接口定义：**

```csharp
public interface ICommand
{
    // 是否可以执行（控制按钮的IsEnabled状态）
    bool CanExecute(object? parameter);

    // 执行命令
    void Execute(object? parameter);

    // CanExecute状态改变事件
    event EventHandler? CanExecuteChanged;
}
```

**基础实现（不使用框架）：**

```csharp
using System;
using System.Windows.Input;

public class SimpleCommand : ICommand
{
    private readonly Action execute;
    private readonly Func<bool>? canExecute;

    public event EventHandler? CanExecuteChanged;

    public SimpleCommand(Action execute, Func<bool>? canExecute = null)
    {
        this.execute = execute;
        this.canExecute = canExecute;
    }

    public bool CanExecute(object? parameter)
    {
        return canExecute?.Invoke() ?? true;
    }

    public void Execute(object? parameter)
    {
        execute();
    }

    // 手动触发CanExecuteChanged
    public void RaiseCanExecuteChanged()
    {
        CanExecuteChanged?.Invoke(this, EventArgs.Empty);
    }
}

// 使用
public class MainViewModel : INotifyPropertyChanged
{
    public ICommand ClickCommand { get; }

    private int count = 0;
    public int Count
    {
        get => count;
        set
        {
            count = value;
            OnPropertyChanged(nameof(Count));
            // 手动通知Command状态改变
            (ClickCommand as SimpleCommand)?.RaiseCanExecuteChanged();
        }
    }

    public MainViewModel()
    {
        ClickCommand = new SimpleCommand(
            execute: () => Count++,
            canExecute: () => Count < 10  // 最多点击10次
        );
    }

    public event PropertyChangedEventHandler? PropertyChanged;
    protected void OnPropertyChanged(string name) =>
        PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(name));
}
```

```xml
<Button Command="{Binding ClickCommand}"
        Content="{Binding Count, StringFormat='点击次数：{0}'}"/>
<!-- 当Count达到10时，按钮自动禁用 -->
```

### 5.2.2 [RelayCommand]源生成器（推荐使用）

CommunityToolkit.Mvvm提供了`[RelayCommand]`源生成器，通过标注方法自动生成ICommand实现，大幅减少样板代码。

**基础用法：**

```csharp
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;
using System.Threading.Tasks;

public partial class MainViewModel : ObservableObject
{
    [ObservableProperty]
    private int count = 0;

    [ObservableProperty]
    private bool isEnabled = true;

    [ObservableProperty]
    private string lastResult = string.Empty;

    [ObservableProperty]
    private bool isLoading = false;

    [ObservableProperty]
    private string errorMessage = string.Empty;

    // 1. 简单Command
    [RelayCommand]
    private void Simple()
    {
        Count++;
        Console.WriteLine($"点击了{Count}次");
    }

    // 2. 带参数Command
    [RelayCommand]
    private void Parameter(string name)
    {
        Console.WriteLine($"你好，{name}");
    }

    // 3. 带返回值Command
    [RelayCommand]
    private string Result()
    {
        return DateTime.Now.ToString("yyyy-MM-dd HH:mm:ss");
    }

    // 4. 异步Command
    [RelayCommand]
    private async Task Async()
    {
        IsLoading = true;
        try
        {
            await Task.Delay(2000);
            Console.WriteLine("异步操作完成");
        }
        catch (Exception ex)
        {
            ErrorMessage = ex.Message;
        }
        finally
        {
            IsLoading = false;
        }
    }

    // 5. 条件Command（CanExecute基于属性）
    private bool CanConditional => IsEnabled;

    [RelayCommand(CanExecute = nameof(CanConditional))]
    private void Conditional()
    {
        Console.WriteLine("条件满足，执行命令");
    }
}
```

**XAML绑定：**

```xml
<StackPanel Spacing="10" Margin="20">
  <!-- 简单Command -->
  <Button Content="简单命令"
          Command="{Binding SimpleCommand}"/>

  <!-- 带参数Command -->
  <Button Content="问候Alice"
          Command="{Binding ParameterCommand}"
          CommandParameter="Alice"/>

  <Button Content="问候Bob"
          Command="{Binding ParameterCommand}"
          CommandParameter="Bob"/>

  <!-- 异步Command -->
  <Button Content="异步操作"
          Command="{Binding AsyncCommand}"
          IsEnabled="{Binding !IsLoading}"/>

  <!-- 条件Command -->
  <Button Content="条件命令"
          Command="{Binding ConditionalCommand}"/>
  <CheckBox Content="启用命令"
            IsChecked="{Binding IsEnabled}"/>
</StackPanel>
```

### 5.2.3 Command vs Vue methods对比

**完整示例对比：**

**Vue实现：**

```vue
<template>
  <div class="controls">
    <button @click="increment" :disabled="count >= 10">
      增加（{{ count }}/10）
    </button>

    <button @click="decrement" :disabled="count <= 0">
      减少
    </button>

    <button @click="reset">
      重置
    </button>

    <button @click="addAmount(5)">
      +5
    </button>

    <button @click="asyncIncrement" :disabled="isLoading">
      异步增加
    </button>

    <p v-if="error" class="error">{{ error }}</p>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const count = ref(0)
const isLoading = ref(false)
const error = ref('')

function increment() {
  count.value++
  error.value = ''
}

function decrement() {
  count.value--
  error.value = ''
}

function reset() {
  count.value = 0
  error.value = ''
}

function addAmount(amount) {
  count.value += amount
}

async function asyncIncrement() {
  isLoading.value = true
  error.value = ''

  try {
    await new Promise(resolve => setTimeout(resolve, 1000))
    count.value++
  } catch (e) {
    error.value = e.message
  } finally {
    isLoading.value = false
  }
}
</script>

<style scoped>
.error {
  color: red;
}

button:disabled {
  opacity: 0.5;
}
</style>
```

**Avalonia实现：**

```xml
<Window xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:vm="clr-namespace:MyApp.ViewModels"
        x:Class="MyApp.Views.MainWindow"
        x:DataType="vm:CounterViewModel"
        Title="Command示例"
        Width="400" Height="300">

  <Window.Styles>
    <Style Selector="TextBlock.error">
      <Setter Property="Foreground" Value="Red"/>
      <Setter Property="Margin" Value="0,10,0,0"/>
    </Style>
  </Window.Styles>

  <StackPanel Margin="20" Spacing="10">
    <!-- 增加 -->
    <Button Command="{Binding IncrementCommand}"
            Content="{Binding Count, StringFormat='增加（{0}/10）'}"/>

    <!-- 减少 -->
    <Button Command="{Binding DecrementCommand}"
            Content="减少"/>

    <!-- 重置 -->
    <Button Command="{Binding ResetCommand}"
            Content="重置"/>

    <!-- 带参数 -->
    <Button Command="{Binding AddAmountCommand}"
            CommandParameter="5"
            Content="+5"/>

    <!-- 异步 -->
    <Button Command="{Binding AsyncIncrementCommand}"
            Content="异步增加"/>

    <!-- 错误信息 -->
    <TextBlock Text="{Binding Error}"
               Classes="error"
               IsVisible="{Binding HasError}"/>
  </StackPanel>

</Window>
```

```csharp
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;
using System.Threading.Tasks;

public partial class CounterViewModel : ObservableObject
{
    [ObservableProperty]
    private int count = 0;

    [ObservableProperty]
    private string error = string.Empty;

    [ObservableProperty]
    private bool isLoading = false;

    public bool HasError => !string.IsNullOrEmpty(Error);

    partial void OnErrorChanged(string value) => OnPropertyChanged(nameof(HasError));

    private bool CanIncrement => Count < 10;

    [RelayCommand(CanExecute = nameof(CanIncrement))]
    private void Increment()
    {
        Count++;
        Error = string.Empty;
    }

    private bool CanDecrement => Count > 0;

    [RelayCommand(CanExecute = nameof(CanDecrement))]
    private void Decrement()
    {
        Count--;
        Error = string.Empty;
    }

    [RelayCommand]
    private void Reset()
    {
        Count = 0;
        Error = string.Empty;
    }

    [RelayCommand]
    private void AddAmount(int amount)
    {
        Count += amount;
    }

    private bool CanAsyncIncrement => !IsLoading;

    [RelayCommand(CanExecute = nameof(CanAsyncIncrement))]
    private async Task AsyncIncrement()
    {
        IsLoading = true;
        Error = string.Empty;

        try
        {
            await Task.Delay(1000);
            Count++;
        }
        catch (Exception ex)
        {
            Error = ex.Message;
        }
        finally
        {
            IsLoading = false;
        }
    }
}
```

### 5.2.4 Command的高级用法

**1. 组合CanExecute条件：**

```csharp
// 多个条件同时满足才能执行
private bool CanSubmit => IsOnline && HasPermission && !IsLoading;

[ObservableProperty]
private bool isOnline = false;

[ObservableProperty]
private bool hasPermission = false;

[RelayCommand(CanExecute = nameof(CanSubmit))]
private void Submit() => Console.WriteLine("提交");
```

**2. Command顺序执行：**

```csharp
// 顺序调用多个异步方法
[RelayCommand]
private async Task LoadAndProcessAsync()
{
    await LoadDataAsync();
    await ProcessDataAsync();
    Console.WriteLine("处理完成");
}
```

**3. 取消Command执行：**

```csharp
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;
using System.Threading;

public partial class MainViewModel : ObservableObject
{
    private CancellationTokenSource? cts;

    [ObservableProperty]
    private int progress = 0;

    [RelayCommand]
    private async Task LongRunningAsync()
    {
        cts = new CancellationTokenSource();

        for (int i = 0; i < 100; i++)
        {
            cts.Token.ThrowIfCancellationRequested();
            await Task.Delay(100, cts.Token);
            Progress = i;
        }
    }

    [RelayCommand]
    private void Cancel()
    {
        cts?.Cancel();
        Console.WriteLine("已取消");
    }
}
```

### 5.2.5 Command最佳实践

**推荐做法：**

```csharp
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;
using System.Threading.Tasks;

public partial class MainViewModel : ObservableObject
{
    // 1. 使用[RelayCommand]，自动管理CanExecute
    [ObservableProperty]
    private bool hasChanges = false;

    [ObservableProperty]
    private bool isValid = false;

    private bool CanSave => HasChanges && IsValid;

    [RelayCommand(CanExecute = nameof(CanSave))]
    private void Save()
    {
        // 保存逻辑
    }

    // 2. 异步操作使用Task返回方法
    [RelayCommand]
    private async Task LoadAsync()
    {
        try
        {
            var data = await _service.GetDataAsync();
            Items = new ObservableCollection<DataItem>(data);
        }
        catch (Exception ex)
        {
            ErrorMessage = ex.Message;
        }
    }

    [ObservableProperty]
    private string errorMessage = string.Empty;
}
```

**避免的做法：**

```csharp
// ❌ 在Command中直接操作UI
[RelayCommand]
private void Bad()
{
    // 不要这样做！
    var button = this.Find<Button>("MyButton");
    button.Content = "Clicked";
}

// ❌ 不处理异常
[RelayCommand]
private async Task BadAsync()
{
    await RiskyOperation();  // 可能抛出异常，没有try/catch
}

// ❌ 阻塞式异步
[RelayCommand]
private void BadBlock()
{
    Task.Delay(1000).Wait();  // 会阻塞UI线程！
}
```

## 5.3 小结

本节深入学习了Command模式：

**ICommand接口**：理解CanExecute和Execute机制
**[RelayCommand]**：使用CommunityToolkit.Mvvm源生成器简化Command创建
**参数传递**：CommandParameter的使用
**异步Command**：返回Task的方法处理异步操作
**条件CanExecute**：基于属性动态控制
**高级用法**：组合条件、顺序执行、取消Command

**关键对比：**

| Vue methods | Avalonia Command | 说明 |
|------------|-----------------|------|
| `methods: { click() {} }` | `[RelayCommand] private void Click() {}` | 无参命令 |
| `@click="fn(arg)"` | `CommandParameter="arg"` | 带参命令 |
| `:disabled="condition"` | `[RelayCommand(CanExecute = nameof(CanXxx))]` | 条件启用 |
| `async fn()` | `[RelayCommand] private async Task FnAsync() {}` | 异步命令 |
| `try/catch` | `try/catch` 内置于命令方法 | 异常处理 |
