# 第五章：事件处理——从@click到Command

在Vue中，我们使用`@click`、`@input`等指令来处理DOM事件。在Avalonia中，事件处理有两种方式：传统的Code-behind（类似HTML的onclick属性）和MVVM推荐的Command模式。本章将深入讲解这两种方式，帮助你理解Avalonia的事件处理机制。

## 5.1 事件绑定

### 5.1.1 事件处理器对比

**Vue的事件处理：**

```vue
<template>
  <div>
    <!-- 内联事件处理器 -->
    <button @click="count++">点击 {{ count }}</button>

    <!-- 方法事件处理器 -->
    <button @click="handleClick">点击我</button>

    <!-- 传递参数 -->
    <button @click="handleDelete(item.id)">删除</button>

    <!-- 访问事件对象 -->
    <input @keyup.enter="submit" />

    <!-- 多个事件 -->
    <button
      @click="onClick"
      @mouseenter="onMouseEnter"
      @mouseleave="onMouseLeave"
    >
      悬停并点击
    </button>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const count = ref(0)

function handleClick() {
  alert('按钮被点击了！')
}

function handleDelete(id) {
  console.log('删除：', id)
}

function submit(event) {
  console.log('按键：', event.key)
}

function onClick() {
  console.log('点击')
}

function onMouseEnter() {
  console.log('鼠标进入')
}

function onMouseLeave() {
  console.log('鼠标离开')
}
</script>
```

**Avalonia的事件处理（Code-behind方式）：**

```xml
<!-- MainWindow.axaml -->
<Window xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        x:Class="MyApp.MainWindow"
        Title="事件处理示例"
        Width="400" Height="300">

  <StackPanel Margin="20" Spacing="10">
    <!-- Click事件 -->
    <Button Content="点击我"
            Click="OnButtonClick"/>

    <!-- 传递参数（使用Tag） -->
    <Button Content="删除"
            Tag="{Binding ItemId}"
            Click="OnDeleteClick"/>

    <!-- 键盘事件 -->
    <TextBox KeyDown="OnKeyDown"
             PlaceholderText="按Enter键提交"/>

    <!-- 鼠标事件 -->
    <Border Background="LightBlue"
            PointerEntered="OnPointerEntered"
            PointerExited="OnPointerExited"
            Height="50">
      <TextBlock Text="悬停在这里"
                 HorizontalAlignment="Center"
                 VerticalAlignment="Center"/>
    </Border>
  </StackPanel>

</Window>
```

```csharp
// MainWindow.axaml.cs (Code-behind)
using Avalonia.Controls;
using Avalonia.Input;
using Avalonia.Interactivity;
using System;

public partial class MainWindow : Window
{
    public MainWindow()
    {
        InitializeComponent();
    }

    // Click事件处理器
    private void OnButtonClick(object? sender, RoutedEventArgs e)
    {
        Console.WriteLine("按钮被点击了！");
        // 在Code-behind中可以直接访问控件
        // 但这不符合MVVM模式（仅用于简单场景）
    }

    // 传递参数
    private void OnDeleteClick(object? sender, RoutedEventArgs e)
    {
        if (sender is Button button && button.Tag is int id)
        {
            Console.WriteLine($"删除：{id}");
        }
    }

    // 键盘事件
    private void OnKeyDown(object? sender, KeyEventArgs e)
    {
        if (e.Key == Key.Enter)
        {
            Console.WriteLine("按下Enter键");
        }
    }

    // 鼠标进入
    private void OnPointerEntered(object? sender, PointerEventArgs e)
    {
        if (sender is Border border)
        {
            border.Background = Avalonia.Media.Brushes.LightGreen;
        }
    }

    // 鼠标离开
    private void OnPointerExited(object? sender, PointerEventArgs e)
    {
        if (sender is Border border)
        {
            border.Background = Avalonia.Media.Brushes.LightBlue;
        }
    }
}
```

### 5.1.2 常用事件对照表

| Vue事件 | Avalonia事件 | 说明 |
|---------|------------|------|
| `@click` | `Click` | 点击 |
| `@dblclick` | `DoubleTapped` | 双击 |
| `@mousedown` | `PointerPressed` | 鼠标按下 |
| `@mouseup` | `PointerReleased` | 鼠标释放 |
| `@mouseenter` | `PointerEntered` | 鼠标进入 |
| `@mouseleave` | `PointerExited` | 鼠标离开 |
| `@mousemove` | `PointerMoved` | 鼠标移动 |
| `@keydown` | `KeyDown` | 键盘按下 |
| `@keyup` | `KeyUp` | 键盘释放 |
| `@input` | `TextChanged` (TextBox) | 输入变化 |
| `@change` | `SelectionChanged` (ComboBox) | 值改变 |
| `@focus` | `GotFocus` | 获得焦点 |
| `@blur` | `LostFocus` | 失去焦点 |
| `@submit` | 无直接对应 | 使用Command |
| `@scroll` | `ScrollChanged` | 滚动 |

### 5.1.3 事件参数对比

**Vue的事件对象：**

```vue
<script setup>
function handleClick(event) {
  console.log('事件类型：', event.type)       // 'click'
  console.log('目标元素：', event.target)      // DOM元素
  console.log('当前元素：', event.currentTarget)
  console.log('鼠标位置：', event.clientX, event.clientY)
  console.log('按键：', event.ctrlKey, event.shiftKey)

  // 阻止默认行为
  event.preventDefault()

  // 阻止冒泡
  event.stopPropagation()
}
</script>
```

**Avalonia的事件参数：**

```csharp
using Avalonia.Input;
using Avalonia.Interactivity;

// Click事件参数
private void OnClick(object? sender, RoutedEventArgs e)
{
    // sender：触发事件的控件
    if (sender is Button button)
    {
        Console.WriteLine($"按钮内容：{button.Content}");
    }

    // e：事件参数
    Console.WriteLine($"事件类型：{e.RoutedEvent?.Name}");
    Console.WriteLine($"事件源：{e.Source}");

    // RoutedEventArgs特有属性
    e.Handled = true;  // 标记事件已处理（类似stopPropagation）
}

// 鼠标事件参数
private void OnPointerPressed(object? sender, PointerPressedEventArgs e)
{
    // 获取鼠标位置
    var position = e.GetPosition(sender as Control);
    Console.WriteLine($"位置：{position.X}, {position.Y}");

    // 获取鼠标按键
    var pointerPoint = e.GetCurrentPoint(sender as Control);
    if (pointerPoint.Properties.IsLeftButtonPressed)
    {
        Console.WriteLine("左键按下");
    }

    // 键盘修饰键
    var modifiers = e.KeyModifiers;
    if (modifiers.HasFlag(KeyModifiers.Control))
    {
        Console.WriteLine("Ctrl键按下");
    }
}

// 键盘事件参数
private void OnKeyDown(object? sender, KeyEventArgs e)
{
    Console.WriteLine($"按键：{e.Key}");          // Key.Enter
    Console.WriteLine($"键码：{e.KeySymbol}");    // "Enter"

    // 修饰键
    if (e.KeyModifiers.HasFlag(KeyModifiers.Control))
    {
        Console.WriteLine("Ctrl + " + e.Key);
    }
}
```

### 5.1.4 Code-behind vs 纯XAML

**Code-behind方式（传统）：**

```xml
<!-- 不推荐：违反MVVM -->
<Button Click="OnButtonClick" Content="点击我"/>
```

```csharp
// MainWindow.axaml.cs
private void OnButtonClick(object? sender, RoutedEventArgs e)
{
    // 直接操作UI，难以测试
    resultTextBlock.Text = "按钮被点击了";
}
```

**Command方式（MVVM推荐）：**

```xml
<!-- 推荐：符合MVVM -->
<Button Command="{Binding ClickCommand}"
        Content="点击我"/>
```

```csharp
// ViewModel
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;

public partial class MainViewModel : ObservableObject
{
    [ObservableProperty]
    private string result = string.Empty;

    [RelayCommand]
    private void Click()
    {
        // 业务逻辑，可测试
        Result = "按钮被点击了";
    }
}
```

**对比总结：**

| 方式 | 适用场景 | 优点 | 缺点 |
|-----|---------|------|------|
| **Code-behind** | 简单UI交互、动画 | 直接、简单 | 难以测试、违反MVVM |
| **Command** | 业务逻辑 | 可测试、符合MVVM | 代码量稍多 |
| **事件转Command** | 复杂事件（如鼠标移动） | 灵活 | 需要转换器 |

### 5.1.5 事件转Command（高级场景）

有些事件（如PointerMoved）不能直接用Command，需要转换：

**方式1：使用Interaction（推荐）：**

```xml
<!-- 安装NuGet包：Avalonia.Xaml.Interactions -->
<Window xmlns:i="clr-namespace:Avalonia.Xaml.Interactivity;assembly=Avalonia.Xaml.Interactivity"
        xmlns:ia="clr-namespace:Avalonia.Xaml.Interactions.Core;assembly=Avalonia.Xaml.Interactions">

  <Border Background="LightBlue">
    <i:Interaction.Behaviors>
      <!-- 将事件转换为Command -->
      <ia:EventTriggerBehavior EventName="PointerEntered">
        <ia:InvokeCommandAction Command="{Binding MouseEnterCommand}"/>
      </ia:EventTriggerBehavior>

      <ia:EventTriggerBehavior EventName="PointerExited">
        <ia:InvokeCommandAction Command="{Binding MouseExitCommand}"/>
      </ia:EventTriggerBehavior>
    </i:Interaction.Behaviors>

    <TextBlock Text="悬停测试"/>
  </Border>

</Window>
```

```csharp
// ViewModel
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;

public partial class MainViewModel : ObservableObject
{
    [ObservableProperty]
    private IBrush backgroundColor = Brushes.LightBlue;

    [RelayCommand]
    private void MouseEnter()
    {
        BackgroundColor = Brushes.LightGreen;
    }

    [RelayCommand]
    private void MouseExit()
    {
        BackgroundColor = Brushes.LightBlue;
    }
}
```

**方式2：自定义Attached Property：**

```csharp
// 附加属性（Attached Property）
public static class EventCommand
{
    public static readonly AttachedProperty<ICommand> CommandProperty =
        AvaloniaProperty.RegisterAttached<Control, ICommand>(
            "Command",
            typeof(EventCommand));

    public static ICommand GetCommand(Control element)
    {
        return element.GetValue(CommandProperty);
    }

    public static void SetCommand(Control element, ICommand value)
    {
        element.SetValue(CommandProperty, value);
        element.PointerEntered += (s, e) =>
        {
            var command = GetCommand(element);
            if (command?.CanExecute(null) == true)
            {
                command.Execute(null);
            }
        };
    }
}
```

```xml
<!-- 使用 -->
<Border local:EventCommand.Command="{Binding MouseEnterCommand}"
        Background="LightBlue">
  <TextBlock Text="悬停测试"/>
</Border>
```
